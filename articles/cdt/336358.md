[大佳说｜有些市场可以假装努力，但外汇不会陪你演戏。](https://chinadigitaltimes.net/chinese/712217.html)
------

<p>9月的数据不重要吗？</p><p>&nbsp; &nbsp;对于有些散户主导的，羊群效应极重，主打一个预期博弈的市场来说，或许9月的数据并不重要。</p><div style="width:42%;float:right;padding-left:20px"><div class="su-spoiler su-spoiler-style-fancy su-spoiler-icon-chevron-circle su-spoiler-closed" data-scroll-offset="0" data-anchor-in-url="no"><div class="su-spoiler-title" tabindex="0" role="button"><span class="su-spoiler-icon"></span>CDT 档案卡</div><div class="su-spoiler-content su-u-clearfix su-u-trim"><strong>标题：</strong>大佳说｜有些市场可以假装努力，但外汇不会陪你演戏。<br><strong>作者：</strong><a href="https://chinadigitaltimes.net/space/大佳说" target="_blank">大佳说</a><br><strong>发表日期：</strong>2024.10.15<br><strong>来源：</strong><a href="https://mp.weixin.qq.com/s/3wcJHPtZkgyDlcu1Zt3lPA" target="_blank">微信公众号“大佳说”</a><br><strong>主题归类：</strong><a href="https://chinadigitaltimes.net/space/中国经济" target="_blank">中国经济</a><br><strong>CDS收藏：</strong><a href="https://chinadigitaltimes.net/space/%E5%85%AC%E6%B0%91%E9%A6%86" target="_blank" rel="noopener">公民馆</a><br><strong>版权说明：</strong>该作品版权归原作者所有。中国数字时代仅对原作进行存档，以对抗中国的网络审查。<a href="https://chinadigitaltimes.net/chinese/copyright">详细版权说明</a>。</div></div></div><p>&nbsp; &nbsp;但对于经济生活中的一个又一个水灵灵的个体来说，数据很重要。</p><p>&nbsp; &nbsp; 先看一下，已经披露的核心的9月数据。</p><p>&nbsp; &nbsp; 1，M1。</p><p>&nbsp; &nbsp; 在去年跌幅已经很低的情况下，跌幅继续扩大。</p><p>&nbsp; &nbsp; 同比下跌7.4%。连续六个月低于水平线以下。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/10/post-712217-670eba6c391d6.png" alt="img"></p><p>&nbsp; &nbsp;2，M2</p><p>&nbsp; &nbsp; 9月份，已经披露的唯一好转的数据。</p><p>&nbsp; &nbsp; 同比增长6.8%。上个月是6.3%。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/10/post-712217-670eba6c616ca.png" alt="img"></p><p>&nbsp; &nbsp; 当然，这次好转的主要原因是，国庆前股市大涨。发生了比较猛烈的银证转账和理财赎回。</p><p>&nbsp; &nbsp; 排除这个因素，M2依然是很惨烈的。</p><p>&nbsp; &nbsp; 3，信贷与社融</p><p>&nbsp; &nbsp; 1）新增人民币贷款15900亿元，市场预期17450亿元，前值9000亿元。</p><p>&nbsp; &nbsp;2）社会融资规模37604亿元，市场预期35167亿元，前值30311亿元。</p><p>&nbsp; &nbsp;具体分项，咱们就不看了。信贷不及预期，但是社融超预期。</p><p>&nbsp; &nbsp;原因无它，政府发债和新增票据。</p><p>&nbsp; &nbsp; 4，居民贷款</p><p>&nbsp; &nbsp;&nbsp;9月新增居民贷款5000亿元，继续同比少增3585亿元。其中短期2700亿元、中长期2300亿元，分别同比少增515亿元、3170亿元，二者均为近五年同期最低。</p><p>&nbsp; &nbsp; 以上是昨天披露的金融数据。</p><p>&nbsp; &nbsp; 我们继续看。</p><p>&nbsp; &nbsp; 5，cpi和ppi</p><p>&nbsp; &nbsp; cpi同比增长0.4%，环比增长0%。</p><p>&nbsp; &nbsp;细分来看：9月份，全国居民消费价格环比持平。其中，城市下降0.1%，农村上涨0.1%；食品价格上涨0.8%，非食品价格下降0.2%；消费品价格上涨0.2%，服务价格下降0.3%。</p><p>&nbsp; &nbsp; 再细分来看：</p><p>&nbsp; &nbsp;&nbsp;细分数据中，鲜菜价格飙升22.9%，贡献了约0.48个百分点的CPI增幅。</p><p>&nbsp; &nbsp; 蔬菜上涨了接近30%，如果把蔬菜排除掉，那么同比就是负增长。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/10/post-712217-670eba6c8d2b4.png" alt="img"></p><p>&nbsp; &nbsp; 也即，通缩吗？的确通缩。</p><p>&nbsp; &nbsp; 但是这个通缩是生必需品，尤其是食物的通胀，而非必需品以及资产价格的通缩。</p><p>&nbsp; &nbsp; ppi在去年同比已经大比例负增长的情况下，继续深度水下。</p><p>&nbsp; &nbsp; 突出一个惨绝人寰。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/10/post-712217-670eba6caaac1.png" alt="img"></p><p>&nbsp; &nbsp; 还有一个数据被严重忽视了。但可能会给本来就已经不富裕的经济雪上加霜。</p><p>&nbsp; &nbsp; 那就是出口。</p><p>&nbsp; &nbsp; 6，进出口数据</p><p>&nbsp; &nbsp;&nbsp;中国9月出口（以美元计价）同比增长2.4%，预期增5.9%，前值增8.7%；进口增长0.3%，预期增1.2%，前值增0.5%；贸易顺差817.1亿美元，前值910.2亿美元。</p><p>&nbsp;</p><p>&nbsp; &nbsp; &nbsp;大幅度不及预期的原因：欧美整体需求回落。当然汇率升值也有那么一点点原因。这是，已经知道的本年出口增速最低月份。但并不是确切的最低月份。因为今年还有三个月。</p><p>&nbsp; &nbsp; 这份数据出来之后，有些市场假装漠不关心，有些市场开始边打边退，有些市场突然恍然大悟，还有些市场，非常不给面子的我行我素。</p><p>&nbsp; &nbsp;</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/10/post-712217-670eba6cce5fb.png" alt="img"></p><p>&nbsp; &nbsp; 常识告诉我们，在这个世界上，有一种无法战胜的力量，叫做经济规律。</p><p>&nbsp; &nbsp;毫无疑问，人类最伟大的智慧，就是等待与希望。</p><p>&nbsp; &nbsp; 但这一切，都有一个前提，就是你能活下去。只有活下去，你才能够等到希望这一天。</p><p>&nbsp; &nbsp; 日子虽然很难过，但日子还是要照过。最关键的是，你不能再去把不多的存粮拿去无端挥霍。余粮真的不多了，真经不起挥霍了。</p><p>&nbsp; &nbsp; 以上。</p><p>&nbsp; &nbsp;作者：皑（ai）大佳</p><div class="addtoany_share_save_container addtoany_content addtoany_content_bottom"><div class="a2a_kit a2a_kit_size_32 addtoany_list"
