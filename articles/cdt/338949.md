抓捕；迫害709；维权 + 任全牛；律师 + 张展；抓捕 + 709</p></blockquote><p>2024年7月9日是中国大规模抓捕维权律师，即“709大镇压”九周年纪念日，包括中国人权律师团在内的全球多个人权组织开展了纪念活动。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/07/id13776423-attorney-combo1-450x300-1.jpg" alt="img"></p><p>“709大镇压”又称“中国709维权律师大抓捕事件”或“709大抓捕”，指的是2015年7月9日起中华人民共和国公安当局在中国多达23个省份大规模逮捕、传唤、刑事拘留、带走、失联、约谈了超过300名律师、人权捍卫者、民间维权人士、上访民众及其亲属的事件。部分人士被强迫失踪，部分人士被判重刑。一些人后来被释放，大约还有10多人仍然在狱中。</p><blockquote><p>709维权律师大抓捕预示着中国对公民社会打压力度手段越来越残忍，其打击是摧毁性的。中国大约有50万律师，从事维权、人权辩护的律师不到1000人，这个比例不是很大的，但依然无法见容于中共。                        —— 滕彪</p></blockquote><p>包龙军是一位长期的法律维权活动者。他与妻子人权律师王宇以及失踪的人权律师王全璋都曾为北京锋锐律师事务所工作过。包龙军是“709大镇压”中第一位失踪的人士。</p><p>2015年7月9日，他与儿子包卓轩原本于北京机场准备登上去往后者即将就学的澳大利亚的飞机，而被警方从机场带走。当晚，警察也搜捕了他们在北京的家，并且将他的妻子绑架。他们夫妻都被置于指定居所监视居住，并持续被关押一年多后才最终获得自由。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/07/cats-600x381-1.jpg" alt="img"></p><p>更多阅读：<a href="https://chinadigitaltimes.net/space/%E4%B8%AD%E5%9B%BD709%E7%BB%B4%E6%9D%83%E5%BE%8B%E5%B8%88%E5%A4%A7%E6%8A%93%E6%8D%95%E4%BA%8B%E4%BB%B6" title="中国数字空间：中国709维权律师大抓捕事件">中国数字空间：中国709维权律师大抓捕事件</a></p><p>在该事件九周年的纪念活动上，“709大镇压”的受害人赵威表示，人类与強权的斗争就是记忆与遗忘的斗争。她说：“纪念709大镇压，不停的把中共做的这些违法犯罪的事情给告诉更多的人，让更多的人知道真相，知道这个政权是一个什么样的政权。任何人做了坏事，都不可以不承担责任的。今天全世界都在关注709，每一次纪念709，都是对他们的一次警告。”</p><h3>事件二: 台湾歌手巴奈夺金曲奖，发言突提六四</h3><h4>敏感词组：</h4><blockquote><p>35届 + 金曲奖；事件 + 金曲奖；巴奈 + 金曲奖；独裁 + 共产；巴奈 + 发声；勿忘 + 天安门</p></blockquote><p>6月29日，台湾举行第35届金曲奖颁奖典礼。台湾原住民独立音乐女歌手巴奈（Panai）以首张台语专辑《夜婆》拿下台语专辑奖。</p><p>在获奖致词时，巴奈说：“金曲奖35年，但是你们知道吗？天安门事件也是刚好35年，我们都不要忘记。”</p><p><iframe loading="lazy" title="巴奈奪最台上提藍綠 更疾呼勿忘&quot;天安門事件&quot; 中國跳腳! 串流音樂下架 微博秒封&quot;巴奈&quot;關鍵字 ｜記者游任博 郭思妏｜【台灣要聞】20240630｜三立iNEWS" width="1080" height="608" src="https://www.youtube.com/embed/kxLRgTkD80M?feature=oembed" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen=""></iframe></p><p>台湾女歌手巴奈出生于台南，她的父亲是卑南族，母亲是阿美族。多年来她一直为台湾原住民争取合法权益而积极参加社会运动。</p><p>金曲奖颁奖典礼结束后，巴奈在后台对媒体表示：“民主的道路漫长而崎岖，我们压力很大，因为我们不知道是否会被‘更大’的势力欺负。”</p><blockquote><p>“我之所以在台上提到（天安门事件）是因为台湾的民主是一个我们所有人都需要珍惜的过程；我们的自由和言论自由，需要我们来守护。”</p></blockquote><p>巴奈在颁奖仪式上触碰两岸议题引起大量中国网民的不满。很多中国网民在Instagram巴奈账号下炮轰：</p><blockquote><p>“做音乐初衷就不纯粹，非得扯上政治，到时候妳不准领身分证”</p><p>“自己是过嘴瘾了，把整个奖害了”</p><p>“解放台湾势在必行！”</p></blockquote><p>不过也有人对巴奈释出善意：“非常期待能在大陆看到您的演出，有机会、有心力的话，很谢谢您的创作！”</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2024/07/504222362d8e5fa7cf5deec49646de4a.webp" alt="img"></p><h3>事件三: 中国经济负面消息不断</h3><h4>敏感词组：</h4><blockquote><p>中国经济 + 彻底崩溃；中国 + 腹背受敌；经济 + 腹背受敌；繁华渐逝；中国经济 + 陷入 + 资产负债表；中国经济 + 彻底崩溃；三中全会 + 中国经济无解药</p></blockquote><p>近期，中文财经圈在热议一篇长文，其题目是《繁华渐逝：腹背受敌的中国经济》，文章广传两天，但稍后就遭到了全网删除。《繁华渐逝》是3万字的长文，加上图表和参考书目，有50页之厚。尽管如此，中文互联网上对于该文章，及其涉及中国经济问题的讨论，还没有歇止。</p><p><a href="https://chinadigitaltimes.net/chinese/709477.html" title="【404文库】丹江湖上钓鱼翁｜繁华渐逝：腹背受敌的中国经济（3万字长文）">【404文库】丹江湖上钓鱼翁｜繁华渐逝：腹背受敌的中国经济（3万字长文）</a></p><p>稍早，美国总统乔·拜登在接受《时代》杂志专访时，质疑了中国经济蓬勃发展的说法，并说人们认为北京越来越强大的说法并不准确。</p><blockquote><p>“这个经济体怎么增长？他们的经济处于崩溃边缘。还有人说他们的经济在蓬勃发展？别扯了。” 拜登补充道。</p></blockquote><p>时事评论员章家敦在接受美国之音采访时认同拜登所说的“中国经济正处于崩溃的边缘”。他认为，中国经济模式已经枯竭。这个国家经历他们自己的2008。当时，中国领导人不愿国家陷入衰退，所以，他们采用了历史上最大的规模的债务刺激计划，现在是他们还债的时候了，但他们没有办法。</p><p>章家敦认为目前中国的政府债务占GDP比率在350%左右，甚至更高。根据世界银行的统计，2022年美国债务占国民生产总值的比例为115.7%，日本为216.5%，而到了中国则是“无此地区数据”。</p><p>但是也有不同的声音，彼得森国际经济研究所高级研究员加里·哈夫鲍尔(Gary Hufbauer)并不认同该说法。</p><blockquote><p>“中国的经济2024年将增长5%，2025年和2026年将增长至少4%。中国没有通胀。它还有足够的财政空间来处理房地产市场的问题，”</p></blockquote><p>根据中国数字时代编辑的整理，在CDT 404文库2024年5月至今收录的被屏蔽的46篇文章中，有14篇事关中国经济的讨论。</p><h4>详见<a href="https://chinadigitaltimes.net/chinese/404-articles-archive" title="404文库专页">404文库专页</a></h4><p>可以察觉的是，中国政府并不希望自己的百姓亦或是外人讨论他们当前的经济状况，以及执政者的处境。</p><p><img decoding="async"
