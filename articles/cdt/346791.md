[【立此存照】继Clash for Windows停更之后，Clash内核等多个项目今日也被删除](https://chinadigitaltimes.net/chinese/701784.html)
------

档案卡</div><div class="su-spoiler-content su-u-clearfix su-u-trim"><strong>标题：</strong>继Clash for Windows停更之后，Clash内核等多个项目今日也被删除<br><strong>作者：</strong><a href="https://web.archive.org/web/https://github.com/Dreamacro/clash" target="_blank">Github</a><br><strong>主题归类：</strong><a href="https://chinadigitaltimes.net/space/翻墙" target="_blank">翻墙</a><br><strong>CDS收藏：</strong><a href="https://chinadigitaltimes.net/space/%E5%85%AC%E6%B0%91%E9%A6%86" target="_blank" rel="noopener">公民馆</a><br><strong>版权说明：</strong>该作品版权归原作者所有。中国数字时代仅对原作进行存档，以对抗中国的网络审查。<a href="https://chinadigitaltimes.net/chinese/copyright">详细版权说明</a>。</div></div></div><p>11月3日，被广大网民熟知的开源翻墙工具 Clash 内核及多个客户端项目在 GitHub上被删除或归档。这一事件在昨日（11月2日）Clash 客户端 Clash for Windows 宣布停更并删库后再次引发网民关注。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2023/11/image-1698926711836.png" alt="img"><span style="font-size: 0.8em;color: #666;display: block;text-align: center;margin-bottom: 20px;line-height: 22px;">Clash for Windows作者@Fndroid 在个人电报频道发布的声明。</span></p><p>据 Clash 的开发者<a href="https://clash.wiki/">介绍</a>，Clash 是一个跨平台的基于规则的代理工具，在网络和应用程序层运行，支持各种代理和反审查协议，在一些互联网受到严格审查或封锁的国家和地区被广泛采用。Clash 也是目前翻墙机场适配最广泛的代理工具之一，网络上有多个版本的 Clash 教程与<a href="https://jiasupanda.com/clash-how-to">使用指南</a>。</p><p>据中国数字时代读者投稿及编辑核查，目前已知受到影响的项目如下：</p><p>已删库：</p><ul><li><a href="https://github.com/Dreamacro/clash">Clash</a>（代理工具）</li><li><a href="https://github.com/Fndroid/clash_for_windows_pkg">Clash for Windows</a>（Clash 客户端，适用于 Windows )</li><li><a href="https://github.com/Kr328/ClashForAndroid">Clash for Android</a>（Clash 客户端，适用于安卓）</li><li><a href="https://github.com/MetaCubeX/ClashMetaForAndroid">Clash Meta For Android</a>（Clash 客户端，适用于安卓）</li><li><a href="https://github.com/yichengchen/clashX">ClashX</a>（Clash 客户端，适用于 Mac )</li><li><a href="https://github.com/SpongeNobody/Clashy">Clashy</a>（Clash 客户端，适用于 Windows / Mac / Ubuntu ）</li><li><a href="https://github.com/EAimTY/tuic">TUIC协议</a>（基于 QUIC 协议的代理工具）</li></ul><p>已归档：</p><ul><li><a href="https://github.com/zzzgydi/clash-verge">Clash Verge</a>（Clash 客户端，支持 Windows、Mac 和 Linux 系统）</li><li><a href="https://github.com/MetaCubeX/Clash.Meta/tree/Meta">Clash.Meta</a>（基于 Clash 二次开发版本的代理工具）</li></ul><p>在GitHub平台上，“删库”意味着整个代码库被删除，项目不再更新，被访问时会出现404错误；“归档”状态意味着项目不再活跃，也不会进行进一步的更新与维护，但仍然可以被公开访问和查看。</p><p>在上述项目中，Clash Verge 于四天前（10月31日）刚刚发布了v1.3.8版本。TUIC协议的开发者EAimTY今日在个人博客上<a href="https://www.eaimty.com/2023/opensource-project-based-on-hormone">表示</a>，此次停更的决定源于该项目持续带来的“压力、内耗以及安全隐患”，以及自己对受众群体的失望。</p><p><img decoding="async" src="https://chinadigitaltimes.net/chinese/files/2023/11/image-1699046071885.png" alt="file"><br><span style="font-size: 0.8em;color: #666;display: block;text-align: center;margin-bottom: 20px;line-height: 22px;">TUIC作者EAimTY的博客截图。</span></p><p>相关阅读：<a href="https://chinadigitaltimes.net/chinese/701751.html">【网络民议】知名翻墙客户端Clash for Windows删库停更</a></p><div
