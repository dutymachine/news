[李清晨｜张文宏教授推荐发热时服用维生素 C，这个错误我在 14 年前也犯过](https://chinadigitaltimes.net/chinese/690916.html)
------

C，这个错误我在 14 年前也犯过<br> <strong>作者：</strong>李清晨<br> <strong>发表日期：</strong>2022.12.15<br> <strong>来源：</strong><a href="https://mp.weixin.qq.com/s/uYWP0cVJa1M0U-MVP5isEQ" href="https://chinadigitaltimes.net/chinese/copyright">详细版权说明</a>。</div></div></div><p>近几年来，在众多公开表达观点的专业人员当中，张文宏教授可能是最受大家敬重的。</p><p>其他专家什么形象，各位想必也都看到了，按照《孙子兵法》的说法，就是「兵无常势，屎无常形」，有些专家的身段就像屎一样柔软，他们会呈现出哪种形象，完全取决于处在哪段肠子里，在横结肠，他们就是圆柱形，在乙状结肠他们就是 S 型，其变身之快令人咂舌。</p><p>但张教授始终能出于一个学者的良心提出最符合实际情况的医疗建议，我不能说张的全部建议都无懈可击，但至少在我看来他从未刻意对公众说谎。</p><p>维生素 C 这个建议，我最初以为只是他的无心之失，但后来我发现他在不同的场合不止一次提起过，联想到张老师读大学的时间，我意识到这个建议应该是他从教科书里学来的。‍</p><p>我在 2008 年时写过一篇《当医生的宝宝遭遇感冒》，其中也提及了服用一定剂量的维生素 C 可以减轻感冒症状，当时发表在群博客上，在此文的留言区就有网友指出，这其实已经是过时的观点了。</p><p>后来《医生爸爸的 365 夜》出版时也收录了这篇文章，我在后面做了补充说明。</p><p><img src="https://chinadigitaltimes.net/chinese/files/2022/12/post-690916-639bfb1e17a46.png" alt="img"></p><p>这确实是我没有及时更新观念造成的失误，因为相关研究早就结论了，只是国内的教材没有及时跟进，在早期的《内科学》《儿科学》教材里，在急性上呼吸道感染一章，都曾将维生素 C 写进治疗建议中（现在已经没有了）。张老师对维生素 C 的推荐想必也是打这来的。‍</p><p>所以，在近期的全网备药的行动中，维生素 C 泡腾片也是被很多人必须要纳入囊中的药品。‍‍</p><p>维生素 C 对人体当然非常重要，但让它在全世界备受瞩目，以至于到了明显超过必要程度，却是源自一位著名科学家的严重失误。</p><p>鲍林（1901~1994）是 20 世纪最伟大的化学家之一，曾先后获得过两次诺贝尔奖，一次是在 1954 年因阐明了化学键的本质和分子结构的基本原理获诺贝尔化学奖，另一次是在上世纪 60 年代获得了诺贝尔和平奖，由于他在化学领域与和平运动方面的贡献，他获得的荣誉学位和奖项超过 30 个。‍‍‍‍</p><p>1970 年，鲍林出版了《维生素 C 与感冒》，大力提倡用维生素 C 预防和治疗感冒，由于他的巨星科学家这一身份，该书立刻畅销全美，维生素 C 也立即身价倍增遭到抢购，以至于超出了当时厂家的生产能力。‍‍‍‍</p><p>这个发生在 50 年前的景象，大家是不是觉得有点儿似曾相识？</p><p>但鲍林的建议，并没有科学依据，他只是把自己的假说当成了现实，并拒不接受来自医学界的批评。</p><p>我在 14 年前被网友批评指正时，重新做了文献检索，我发现在 2004 年有一篇系统综述收入了所有感冒应用维生素 C 与安慰剂的随机对照研究，结果显示，感冒症状初起后，应用维生素 C 与安慰剂相比，症状在持续时间上没有显著差异。‍‍‍‍‍</p><p>如果鲍林能够重视同时代医学研究人员在维生素 C 方面的研究，也许他就不会那么固执己见了，比如 1967 年英国的蒂勒尔领导的感冒研究小组就对比过口服维生素 C 与安慰剂的区别，结果发现维生素 C 无效。‍</p><p>但鲍林完全无视这些研究，他发挥自己的雄辩天赋，对那些不符合自己假想的研究结论横挑鼻子竖挑眼，什么剂量太小啦，资料分析不当啦，总之，你们都特么是笨蛋，请听我这个百年一遇的天才科学家的才对！</p><p>医学家们对鲍林的狂妄嗤之以鼻，行，你不是说我们原来的研究不严谨么？我们再来几次，看你服不服。</p><p>最后，至少有 16 篇设计周密的对照双盲试验证明，维生素 C 无明显作用。</p><p>这里面还有一个异常有趣的细节，因为盲法的要求，维生素 C 片与安慰剂（乳糖片）在外观和口感上是设计的没有区别的，但有些淘气的志愿者居然试图把药片咬碎——我尝尝啥味儿的（维生素 C 味道酸，人家本来就叫 抗坏血酸嘛），猜猜我吃的到底是维生素 C 还是安慰剂。‍‍‍‍</p><p>结果还真就有 79 人猜中，以至于在这部分人中，维生素 C 似乎显示出了部分效果，也就是说，当人们对一种药物产生信仰时，就可以出现虚假的效果。‍‍</p><p>我们今天已经理解了信仰疗法的产生机制，暗示和自我暗示可以缓解某些疾病的症状，信仰——宗教般狂热的张力会创造出一种非常有利于治疗的精神 状态，但也只有像鲍林这样的超级巨星才有这样的魔力，我要说读书能治感冒恋爱能防新冠，也不可能有人当真啊。‍</p><p>在我们后人看来，这些试验已经毫无疑问地证明鲍林是错的了，再看不出来，那简直就是傻子。</p><p>但鲍林就是不承认，他抨击医学界和药厂完全是为了感冒药的销量才故意抹杀有利于维生素 C 的证据——嘿嘿，这种阴谋论大家是不是也特别熟悉？</p><p>1976 年鲍林再版了《维生素 C 与感冒》，在核心观点不变的基础上，再增加两章专门批评医学界的——不听我的，我就骂死你们。‍</p><p>让你们不信我说的维生素 C 能预防和治疗感冒，我再出一本书说维生素 C 还能治疗癌症呢！‍</p><p>1979 年，鲍林出版《维生素 C 与癌症》，1986 年，鲍林在《怎样才能长寿与健康》一书中，更是把维生素 C 捧上了天，说维生素 C 可以增进健康，有助于防治心脏病、癌症和其他疾病，并且延缓衰老。</p><p>医学界还像以前一样，认认真真做对照试验，对鲍林的胡说八道一一予以迎头痛击。</p><p>我们无法得知晚年鲍林的内心究竟是一个什么状态，他会很痛苦么？还是斗志昂扬地与所有的反对者死战到底？</p><p>但我们需要明白的是，金无足赤人无完人，历史上著名科学家晚年误入歧途的教训比比皆是，鲍林是公认的杰出科学家，英国的 New Scientist 甚至把他列为人类迄今最伟大的 20 位科学家之一，可与伽利略、牛顿和达尔文、爱因斯坦比肩齐名。</p><p>幸运的是，美国的医学界没有被他误导，而是用一个又一个严谨的临床试验研究将他的谬说驳倒。</p><p>遗憾的是，鲍林的很多医学建议早就已经影响到美国之外的很多国家了。</p><p>再伟大的人，也有犯错误的可能，维生素 C 被长期滥用的教训提醒我们，诉诸权威是一种很危险的思维方式，不唯书不唯上只唯实，只有坚持真正的实事求是的科学精神，才能保障人类世界健康发展。</p><p>周公恐惧流言日，王莽谦恭未篡时。向使当初身便死，一生真伪复谁知。也许，有些人一生中最大的错误就是，死晚了。</p><div class="addtoany_share_save_container addtoany_content
