href="https://chinadigitaltimes.net/chinese/copyright">详细版权说明</a>。</div></div></div><p>8月17日，福建省互联网辟谣平台 @福建辟谣 就一篇网文《100名中国人存瑞士银行7.8万亿？都是些什么人》进行辟谣，文章称原文中的“7.8万亿”其实是2017年中国多位富豪的所有资产，并非瑞银存款，实际人数为373名，也并非原文标题中的“100名”…..然而，微博账号 @财经杂志 在转发这篇辟谣文章后引发了网民对于中国权贵/精英阶层海外资产的热议、质疑，最终评论区开启了消息审查、转发功能也被关闭，一千余条网民评论已不可见。</p><blockquote><p>@财经杂志：【官方辟谣网传100名中国人存瑞士银行7.8万亿 ：别再以讹传讹】据福建辟谣，近日，网上热传《100名中国人存瑞士银行7.8万亿？都是些什么人》，引发网民的关注。文章称：瑞士银行公布消息，100位中国人在瑞士银行的存款合计7.8万亿，则平均存款为：78000亿/100=780亿人民币。福建辟谣查证，网传消息不实！网传消息中的“瑞士银行”其实是指瑞银集团（UBS）。据相关资料介绍：它是欧洲最大的金融控股集团，总部位于瑞士苏黎世，被《欧洲货币》从2004至2007年连续四年评为全球最佳私人银行。谣言分析：人民币约7.8万亿元是2017年时中国373名富豪所有资产，不仅仅是瑞银存款。把所有资产说成只有瑞银存款，显然是不恰当的。且把“373名”换成“100名”，这博眼球的意味更浓了。可见谣言就是这样以讹传讹制造出来的。 （福建辟谣）</p><p><img src="https://chinadigitaltimes.net/chinese/files/2022/08/image-1660979136711.png" alt="filev"></p></blockquote><p>@财经杂志 评论区开启审查后的截图：</p><p><img alt="file"><br><img src="https://chinadigitaltimes.net/chinese/files/2022/08/image-1660979231285.png" alt="file"></p><p>目前为止，相关微博话题“#官方辟谣网传100名中国人存瑞士银行7.8万亿”仍在，于是有大量网民转至其他热门消息评论区发表个人观点。</p><p><img src="https://chinadigitaltimes.net/chinese/files/2022/08/image-1660979504292.png" alt="file"><br><img src="https://chinadigitaltimes.net/chinese/files/2022/08/image-1660979831696.png" alt="file"></p><p><strong>以下为中国数字时代编辑摘自网民评论：</strong></p><blockquote><p>幸运涨涨up：一看就是假的，一百人，可能背后就是几个人而已。 // 余音唱晚：一看就是假的，不可能是人民币，应该是瑞郎。 //  无奈一场空： 一看就是假的，怎么可能这么少？</p><p>唱歌治感冒：评论都关了，不禁让我想起许多成语和典故，例如什么欲盖弥彰，此地无银三百两什么的。</p><p>超级替补索圣：不开评论基本就实锤了。</p><p>少帅的野望：鼓励瑞士黑掉这些钱。 //  闲鱼包咸：没收，你信不信被压榨的更狠，几十年的敲骨吸髓，他们能几年拿回来。</p><p>2022围城：毫无疑问这些人肯定会不惜代价阻止武统的。</p><p>三次出发：请官方拿出证据辟谣。</p><p>我的读书小马甲：为了不让这类谣言满天飞，希望全国推行官员财产公示。</p><p>麦田蓑笠翁：嗯，造谣可耻。明明这7.8万亿是373名富豪的存款和资产，楞要说成是100个人的。</p><p>一天一包魔芋爽：贫富差距太大了，还是把共同富裕给删了吧，这样好让我心甘情愿做牛马。</p><p>jcgssong：373与100对于14亿也没啥区别。</p><p>侠影再现：去年瑞士要公开客户名单，某国第一个跳出来反对，自己想吧。</p><p>华山吟游诗人：马云，马化腾有这个钱，我一点不生气，这是人家凭本事挣得，你这钱咋来的，无非是权力给你带来的。</p><p>qd-lucy：前几天瑞士在欧洲第一个跳出来喊话要制裁我们，不是没有根据的。</p><p>JT养肉不长肉：非常有可能，17年之前就有组数据说，把中国全部贪污和非法的钱可以发养老金好几辈人用，之后不知为何就再也看不到这些新闻了，还有4万亿（约莫记得是）入股的新闻也看到过一次就没后话了。</p><p>Fullmiere：幸亏是谣言，现在全国人民情绪很稳定。</p><p>许二诺：大连市副市级干部徐长元涉嫌贪污受贿，被立案调查，经查明，徐长元一案总共涉及2714套房产、46家企业、142辆豪车，总价值约为6500亿。此数据一出，当场震惊了所有人，有人表示，大连市2020年的GDP才7000亿左右，这徐长元的6500亿，岂不是相当于一个市一年的努力。</p><p>发光的妖怪：某些人一个人就存了几百亿，他敢承认吗？</p><p>鲸落星河三千：钱在少数人手里是事实，而且永远不会有先富带动后富，毕竟谁会嫌它多呢，有钱的人赚钱会上瘾的。</p><p>rainieFHY：不是共同富裕嘛，为何把我落下。</p><p>我有长生草：保护领导隐私。</p><p>城北老Young：这些存款人的身份很敏感吗？怎么禁止评论和转发啊？</p><p>如风般自由1013：你们不辟谣，我都不知道这个谣，还有多少这样你们辟过我才知道的谣？</p><p>亚拉那依卡：都不是第一天做中国人，别装外宾好吗？</p><p>陀枪师兄：真正的富豪不在富豪榜上。</p><p>可惜没如果_D：总结有373名富豪在瑞银有不少的存款？</p><p>青木生生不息：评论都不敢开那应该是真的。</p><p>吃火锅用芝麻油：这么说吧，某些学校每次打钱到部门时都会找十个学生的银行卡接收这笔钱，然后再让学生把这钱转给老师统一收起来，所以这一百个人也许不是钱的主人。</p><p>有事问彭叔：从2017年到现在，经过这5年的发展，国内这些富人的资产水平，肯定是远超了当时的7.8万亿，而且能出现在明面上的373名富豪，他们的资产就有7.8万亿，那么那些不能见光的影子富豪呢？这也是为什么那么多人明知道这个是谣言，但还是愿意相信的原因。这个里面是真假都有，所以才难以判断。</p></blockquote><div
