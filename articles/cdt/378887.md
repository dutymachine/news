[轻芒杂志｜朋友还是敌人？从7 篇文章了解塔利班](https://chinadigitaltimes.net/chinese/669688.html)
------

<p>原标题：轻芒封面 | 朋友还是敌人，7 篇文章带你全面了解塔利班</p><p>作者：wanhu</p><p>8月16日，阿富汗塔利班发布消息称，其已控制位于阿富汗首都喀布尔的总统府。从今年 5月开始一路攻城略地到占领首都喀布尔，阿富汗塔利班只用了三个月。在国内社交舆论中，有不少人因为塔利班的反美姿态而对其充满好感，而在7月28日外交部长王毅会见阿富汗塔利班政治委员会负责人巴拉达尔后，因巴拉达尔称中国一直是阿富汗人民值得信赖的好朋友，微博上不少人更是声称塔利班是中国的好朋友。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca08f0c9.png" alt="image"></noscript><img class="lazyload" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca08f0c9.png" alt="image"></p><ul><li><a href="http://www.xinhuanet.com/world/2021-07/28/c\_1127705163.htm" title="新华社 王毅会见阿富汗塔利班政治委员会负责人巴拉达尔">新华社 王毅会见阿富汗塔利班政治委员会负责人巴拉达尔</a></li></ul><p>自然也有很多人对这个言论进行了反驳，举例了不少塔利班曾经的暴行。中文互联网因为塔利班到底是朋友还是敌人而吵作一团，今天的封面文章，我们也想讨论一下塔利班，不是去讨论是否是朋友，而是去看看塔利班到底是谁，曾经做了什么，和大家一起更多地去了解塔利班。</p><h3><strong>与恐怖组织渊源颇深的塔利班</strong></h3><p>「塔利班」一词原本并无政治含义，在阿拉伯语中，「塔利布」意为学生，其复数形式便是「塔利班」。该组织成立于 1994 年，由毛拉·奥马尔召集了他的学生和一些来自巴阿边境难民营的阿富汗流亡青年组成，鉴于这支队伍主要由宗教学生组成，故而被外界称为塔利班。1996 年塔利班开始统治阿富汗，直到 2001 年美军打击基地组织，也间接导致塔利班政权倒台。</p><p>2007 年，塔利班领导人贝图拉·马赫苏德宣布自立门户，成立「巴基斯坦塔利班运动」（Tehrik-e-Taliban，简称 TTP）。至此就有了两个塔利班，阿富汗塔利班（简称阿塔）和巴基斯坦塔利班（简称巴塔），两个组织的性质和目的是截然不同的。</p><p>对于在阿富汗的塔利班组织来说，建立国家是他们一直以来的目标。「文化纵横」的文章对此有详细的讨论。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca0b5bea.png" alt="image"></noscript><img class="lazyload" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca0b5bea.png" alt="image"></p><ul><li>文化纵横 <a href="https://mp.weixin.qq.com/s?__biz=MzA5MjM2NDcwMg==&amp;mid=2664322361&amp;idx=1&amp;sn=e992bfe5330f2b15fb2b5ca7d55716f2&amp;scene=21#wechat_redirect">突访中国的塔利班, 真正目的是什么? 一个特殊智库的揭底</a></li></ul><p>而巴基斯坦塔利班组织则是巴基斯坦政府和国际社会普遍认定的恐怖组织，制造了多起恐怖袭击活动。「纵相新闻」的文章分析了两者的区别。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca0d8a0c.png" alt="image"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca0d8a0c.png" alt="image"></p><ul><li>纵相新闻 <a href="https://mp.weixin.qq.com/s?__biz=MzU1MzkzNjU2MQ==&amp;mid=2247524343&amp;idx=1&amp;sn=4f3423815ce851fc8a27dbe79cc52507&amp;scene=21#wechat_redirect">“两个塔利班”的前世与今生</a></li></ul><p>值得注意的是，巴基斯坦塔利班是国际社会公认的恐怖组织，而阿富汗塔利班也曾因资助恐怖组织「东突伊斯兰运动」而被点名，这个在国务院于 2002 年公布的制裁名单里可以查阅到。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca106e0b.png" alt="image"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca106e0b.png" alt="image"></p><ul><li><a href="http://www.gov.cn/gongbao/content/2002/content\_61790.htm" title="中华人民共和国中央人民政府 关于联合国安理会制裁阿富汗委员会将“东突厥斯坦伊斯兰运动”列入受制裁实体名单的通知">中华人民共和国中央人民政府 关于联合国安理会制裁阿富汗委员会将“东突厥斯坦伊斯兰运动”列入受制裁实体名单的通知</a></li></ul><p>事实上，按照 2016 年颁布的《中华人民共和国反恐怖主义法》中对恐怖组织的定义「为恐怖活动组织、恐怖活动人员、实施恐怖活动或者恐怖活动培训提供信息、资金、物资、劳务、技术、场所等支持、协助、便利的;」，阿富汗塔利班组织即使算不上恐怖组织，也不可能是朋友。</p><ul><li><a href="https://cati.nwupl.edu.cn/dtzx/flfg/1136.htm" title="反恐主义信息网 中华人民共和国反恐怖主义法">反恐主义信息网 中华人民共和国反恐怖主义法</a></li></ul><p>另外，作为武装组织，能在美军的打击下坚持到现在而且还发展壮大，阿富汗塔利班组织的资金来源也颇受关注，外媒《The Conversation》的一篇文章就详细研究了塔利班的收入来源，发现最大收入来源是毒品。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca12a598.png" alt="image"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca12a598.png" alt="image"></p><ul><li>同时hxotnongd <a href="https://mp.weixin.qq.com/s?__biz=MzA4MDMyMjUyMQ==&amp;mid=2447988104&amp;idx=1&amp;sn=caf9ee4ca6db3715f1f44a8627761cc8&amp;scene=21#wechat_redirect">塔利班的资金从哪里来 ｜ 与此同时</a></li></ul><p><a href="https://mp.weixin.qq.com/s?__biz=MzkyNzIwODI3OA==&amp;mid=2247546314&amp;idx=2&amp;sn=e4fccc1edbde059fc0ef39cdb71b6ac7&amp;scene=21#wechat_redirect">有数据显示</a>，自 2001 年以来，阿富汗成了世界最大毒品生产基地和输出国，全球 90% 的罂粟类毒品（鸦片、海洛因）产自这里。</p><h3><strong>作为极端宗教组织的塔利班</strong></h3><p>塔利班组织最显著的特点是其的宗教性，是一场政教合一的组织运动，而且是极端伊斯兰宗教，对于女性和其他宗教的看法尤其极端。在社会生活上，塔利班禁止演奏乐曲、跳舞、观看电影和电视等一切娱乐、体育活动等，《西亚非洲》杂志 2016 年第 3 期《塔利班运动及其伊斯兰实践新探——兼论中东政治伊斯兰的“塔利班化”新动向》一文就从宗教和政治性上探讨了塔利班的的极端性。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca15308b.png" alt="image"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca15308b.png" alt="image"></p><ul><li>西亚非洲 <a href="https://mp.weixin.qq.com/s?__biz=MzI1MzAwNDcwOA==&amp;mid=2652475673&amp;idx=3&amp;sn=94b1ad4ef83c63961edeb9825ca4c4a3&amp;scene=21#wechat_redirect">塔利班运动及其伊斯兰实践新探 ——兼论中东政治伊斯兰的“塔利班化”新动向</a></li></ul><p>而且，在塔利班看来，阿富汗民族的历史就是伊斯兰教传入以来的历史，其他文明由异教徒创造，皈依伊斯兰教之前的历史是「蒙昧时代」。所以，塔利班不顾埃及知名宗教人士的劝阻，炸掉「巴米扬大佛」，并毁掉大量珍贵文物。这也就是很多人所熟知的，发生于 2001 年 3 月份的「巴米扬大佛」事件，公众号「旃檀精舍」在今年 3 月——大佛被毁 20 周年的时候发文介绍了这起惨案。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca17a4f7.png" alt="image"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca17a4f7.png" alt="image"></p><ul><li>旃檀精舍 <a href="https://mp.weixin.qq.com/s?__biz=MjM5NjQ0OTA4Nw==&amp;mid=2652046641&amp;idx=1&amp;sn=351d78095a90daf60ab6e47337653780&amp;scene=21#wechat_redirect">巴米扬之殇，大佛被毁20周年祭</a></li></ul><p>虽然塔利班的发言人之前曾表示，将组建一个「开放、包容的伊斯兰政府」，允许女性独自出门、接受教育和工作。但是就在最近刚被塔利班占领的城市里，女性已经开始重新被要求穿上了罩袍，历史会不会重演，谁都不知道。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca1a37e5.png" alt="image"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/08/post-669688-611b7ca1a37e5.png" alt="image"></p><ul><li>纵相新闻<a href="https://mp.weixin.qq.com/s?__biz=MzU1MzkzNjU2MQ==&amp;mid=2247525149&amp;idx=2&amp;sn=22f7d2234e1fb28d3216143a28c6da2b&amp;scene=21#wechat_redirect">被塔利班占领后的城市：枪炮声平息，妇女重新穿上罩袍</a></li></ul><p>…</p><p>塔利班到底怎么样，或许从上文中的种种报道中能够略知一二。国际问题的复杂之处在于，你没法简单用朋友或者敌人去定义别的国家，事实上也不需要这样去定义。人与人之间除了朋友和敌人，尚且还有其他各种复杂的关系，更不用说国际关系了。</p><p>当然也没有所谓的「敌人的敌人是朋友」——这样一种把关系简单化的说法。</p><p>（完）</p><div
