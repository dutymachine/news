Rezin Garcia/Sun-Times, via Chicago Sun-Times, via Associated Press</figcaption></figure><p>芝加哥大学硕士毕业生郑少雄11月9日在校园附近被劫杀后，国内媒体一篇纪念他的微信公众号<a href="https://mp.weixin.qq.com/s/QZ4AxGHO5b3GY7doobVpUw" title="" rel="noopener noreferrer" target="_blank">文章</a>在朋友圈里刷了屏。郑少雄的一位发小接受采访，回顾了他的成长经历，也讲述了他妈妈在得知芝大有学生出事又联系不上儿子时的那种煎熬。一切祈祷和希望，最后都被领馆的那通电话击碎了。对所有孩子只身在外求学的父母而言，这样的电话都是一场噩梦。</p><p>郑少雄的妈妈不是唯一收到这样的噩耗的中国家长。今年年初的芝大博士生<a href="https://www.nytimes.com/2021/01/10/us/chicago-evanston-shooting-spree-university.html" title="">范轶然</a>、2017年伊利诺伊大学香槟分校的访问学者<a href="https://www.chicagotribune.com/news/breaking/ct-ct-missing-illinois-student-met-20170702-story.html" title="" rel="noopener noreferrer" target="_blank">章莹颖</a>、2016年亚利桑那州立大学的<a href="https://www.nbcnews.com/news/us-news/asu-student-jiang-yue-s-road-rage-killing-sparks-china-n499331" title="" rel="noopener noreferrer" target="_blank">江玥</a>、2014年南加州大学的<a href="https://www.nytimes.com/2014/07/30/us/california-4-teenagers-charged-with-murder-over-the-beating-of-a-graduate-student.html" title="">纪欣然</a>、2013年波士顿大学的<a href="https://www.nytimes.com/2013/04/18/world/asia/china-mourns-the-death-of-student-in-boston-blast.html" title="">吕令子</a>、2012年南加大的<a href="https://www.nytimes.com/2012/05/20/us/2-arrested-in-fatal-shooting-of-students-from-china.html" title="">瞿铭、吴颖</a>，2010年计划入读法学院的<a href="https://gothamist.com/news/woman-confronts-man-who-raped-killed-daughter-i-wish-a-life-of-hell-for-you" title="" rel="noopener noreferrer" target="_blank">姚宇</a>……每一次中国留学生在美国死于非命，他们的家人都会陷入这样的噩梦，也都会在中国家长中引发新一轮的追问：美国是不是太危险了？还能送孩子去那里读书吗？</p><p>为人父母，这样的担心并非凭空而来。孩子在成长过程中难免要经历离家，独自面对陌生的世界和未知的风险。从多年前的<a href="https://www.chinanews.com.cn/gn/2021/03-30/9443981.shtml" title="" rel="noopener noreferrer" target="_blank">南京大学碎尸案</a>到近年的<a href="http://www.xinhuanet.com/politics/2015-12/12/c_128522910.htm" title="" rel="noopener noreferrer" target="_blank">复旦大学投毒案</a>都表明，即使孩子是在中国读大学，家长也并不一定就能高枕无忧。但中国毕竟对枪支有严格管控，美国的<a href="https://www.nytimes.com/spotlight/gun-violence" title="">枪支暴力</a>则已成顽疾，上述九名留学生中有五人都是死于枪下。对于习惯了“深夜街头撸串”这种安全感的中国人，枪比其他任何杀人工具都更惊悚高效；这类随机作案的残暴与突发性，也会让人们觉得，哪怕学习了<a href="https://dps.usc.edu/safety-tips/suspicious-activity/robbery/" title="" rel="noopener noreferrer" target="_blank">逃生技能</a>，仍然可能防不胜防。</p><p>近年来，美国多地反警察暴力示威游行中夹杂<a href="https://www.nytimes.com/article/george-floyd-protests-timeline.html" title="">抢掠</a>乱象，活动人士还呼吁<a href="https://www.nytimes.com/2020/08/04/us/minneapolis-defund-police.html" title="">削减警察经费</a>，加上疫情暴发以来针对亚裔的<a href="https://cn.nytimes.com/usa/20210609/asian-american-attacks/" title="">歧视和仇恨事件</a>频频发生，在隔着大洋雾里看花的中国家长眼中，这一切都令原本就存在的治安问题雪上加霜。疫情以来，美国的谋杀率也的确出现了上升态势。2020年美国<a href="https://www.nytimes.com/2021/09/27/us/fbi-murders-2020-cities.html" title="">死于谋杀的人数</a>比2019年多出近5000名，创下自1960年有记录以来的最大年度增幅，其中77%的谋杀案为枪杀。</p><aside aria-label="companion column"></aside><p>不光是中国人认为美国“水深火热”，美国人自己也对暴力犯罪高企深感惊惧。但要对美国的安全程度做出判断，也不能忽视公众对犯罪趋势常常存在的误读。<a href="https://news.gallup.com/poll/323996/perceptions-increased-crime-highest-1993.aspx" title="" rel="noopener noreferrer" target="_blank">盖洛普</a>的调查发现，不论实际数据如何，选民总感觉犯罪率一年比一年高。一些分析认为这与公众习惯聚焦负面新闻有关。这样说来，中国留学生无辜被害的新闻对中国公众——特别是留学生和家长这个群体的心理冲击大概也会比一般的美国负面新闻更强烈。更何况在这个社交媒体时代，怀消息总是在家长们的朋友圈里传得飞快，也更容易令他们焦虑。</p><p>但事实上，美国的犯罪率从上世纪九十年代开始持续下降，2020年的<a href="https://www.nytimes.com/2021/11/15/us/homicides-america.html?campaign_id=2&amp;emc=edit_th_20211116&amp;instance_id=45497&amp;nl=todaysheadlines&amp;regi_id=50740216&amp;segment_id=74502&amp;user_id=11df0993621e3f552d0c6df6eea2c4aa" title="">谋杀率</a>为每10万人6.5起，虽然较上一年有显著上升，但相比1991年峰值期的每10万人<a href="https://www.brennancenter.org/our-work/research-reports/crime-trends-1990-2016" title="" rel="noopener noreferrer" target="_blank">9.8起</a>仍低得多。具体到<a href="https://opendoorsdata.org/fact_sheets/china/" title="" rel="noopener noreferrer" target="_blank">中国留学生</a>，疫情前这个群体的人数已经达到37万，2020年有所下降，但也保持在近32万，而近十年在美国被杀害的中国留学生仍然屈指可数。</p><p>当然，统计数字勾勒出的整体图景，对成为谋杀率分子的个体来说是没有意义的。在芝加哥，那个下午，那个时刻，随着那声枪响，世界失去了一个踌躇满志的年轻人。在中国，在四川乐山，一个单亲妈妈接到了领馆打来的电话，她失去了自己唯一的孩子。对郑少雄的亲朋来说，这就是一切。</p><aside
