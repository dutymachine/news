[中国上市公司回应俄乌战火是否波及公司经营 中交建称在乌克兰无在建项目](https://cn.reuters.com/article/china-listed-firms-ukraine-cccc-0225-idCNKBS2KU15J)
------

<div><i>2022-02-25T11:03:44Z</i></div><p>路透上海2月25日 - 俄罗斯和乌克兰开战，陆续有中国A股上市公司在投资者互动平台回应在俄罗斯或乌克兰的经济利益是否受俄乌冲突影响。</p><p>央企基建巨头--中国交建在互动平台表示，公司在乌克兰没有在建项目，在俄罗斯有少量在建项目，占公司2021年中期在手未完工合同额的0.0008%，对公司整体影响较小。中国交建A股周五收跌4.8%，已连续两个交易日收低。</p><p>本地媒体引述 <a href="https://www.jiemian.com/article/7142659.html">here</a>水利水电工程承包企业--中国电建指出，该公司在乌克兰有一个项目，但目前还没有开工，所以整体影响非常小。中国电建股价今日收盘持平。</p><p>保温杯生产商--嘉益股份则称 <a href="http://irm.cninfo.com.cn/ircs/question/questionDetail?questionId=1125092927302090752">here</a>，有保温杯产品销往俄罗斯，但数量较少。</p><p>中曼石油表示，公司目前海外项目包括俄罗斯项目都在正常运营，俄乌冲突未对公司海外项目及生产经营造成影响。</p><p>风力发电机配套塔架制造商--泰胜风能表示 <a href="http://irm.cninfo.com.cn/ircs/question/questionDetail?questionId=1124937294064238592">here</a>，公司销往乌克兰的订单目前没有违约迹象；轴流压缩机供应商--陕鼓动力亦称 <a href="http://sns.sseinfo.com/search.do?keyword=601369&amp;keywordEnd=601369&amp;type=1&amp;sdate=&amp;edate=">here</a>，目前公司在俄促销项目、在执行项目均未受到俄乌关系影响。</p><p>基础化工原料供应商--诚志股份也提到 <a href="http://irm.cninfo.com.cn/ircs/question/questionDetail?questionId=1124808669185712128">here</a>，目前公司暂未受到俄乌局势紧张带来的直接影响，生产经营一切正常。</p><p>俄罗斯和乌克兰开战第二天，乌克兰总统泽连斯基周五发誓要留在首都基辅，他在推特上写道“我们需要(一个)反战联盟。”欧美国家对俄罗斯加码制裁，美国和其他北约成员国已向乌克兰提供了军事援助。(完)</p>
