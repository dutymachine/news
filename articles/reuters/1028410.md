[中国五年LPR 21个月来首降但幅度仅5bp 凸显地产仍在托底非刺激](https://cn.reuters.com/article/china-cen-lpr-rea-stimulating-0120-idCNKBS2JU073)
------

<div><i>2022-01-20T02:36:40Z</i></div><p>路透北京1月20日 - 紧随中期借贷便利（MLF）操作利率下调步伐，中国1月LPR（贷款市场报价利率）报价中一年期利率在上月下调5bp（基点）后，再度下调10bp至3.70%，同时五年期以上品种利率亦录得21个月来首次下调，不过下调幅度仅在5bp。</p><p>市场人士认为，一年期报价下调幅度符合预期，且与MLF降息幅度同步；而五年期下调幅度略小，一方面表明报价行有利差收益考量，另一方面亦说明房地产仍是托底而非刺激。</p><p>“一年符合预期，五年也在情理之中，‘房住不炒’定位没变，刺激还谈不上，主要还是托底的意思，”上海一私募研究人士称。</p><p>路透调查显示，受访市场人士一致预计本月LPR会调降，其中认为一年和五年均下调10个基点（bp）的占比达63%，同时预计一年下调10bp五年下调5bp的人士占比近21%</p><p>新年后对于中国政策利率降息的预期明显升温，央行周一亦“顺应市场呼声”，时隔21个月调降MLF和逆回购利率，且幅度均达此前预期的上限10个bp。面对经济下行压力加剧，以及房地产风险仍在持续释放的双重考验，政策应对明显发力。</p><p>中国央行2019年8月推出LPR新机制，一年期品种当年内三次下调，累计降幅为16个基点（bp）至4.15%；新增的五年期以上品种2019年8月首次报在4.85%，随后在当年11月下调一次至4.80%。2020年，一年和五年期LPR各下调两次，其中一年期累计下调30个bp，五年期累计下调15bp。2021年12月，央行20个月来首次下调一年LPR报价5bp，五年保持不变。</p><p>以下为LPR新机制报价的各期情况：</p><p>日期 一年 五年以上</p><p>利率 变动 利率 变动</p><p>2022年1月20日 3.70% -10bp 4.60% -5bp</p><p>2021年12月20日 3.80% -5bp 4.65% 0</p><p>11月22日 3.85% 0 4.65% 0</p><p>10月20日 3.85% 0 4.65% 0</p><p>9月22日 3.85% 0 4.65% 0</p><p>8月20日 3.85% 0 4.65% 0</p><p>7月20日 3.85% 0 4.65% 0</p><p>6月21日 3.85% 0 4.65% 0</p><p>5月20日 3.85% 0 4.65% 0</p><p>4月20日 3.85% 0 4.65% 0</p><p>3月22日 3.85% 0 4.65% 0</p><p>2月20日 3.85% 0 4.65% 0</p><p>1月20日 3.85% 0 4.65% 0</p><p>2020年12月21日 3.85% 0 4.65% 0</p><p>11月20日 3.85% 0 4.65% 0</p><p>10月20日 3.85% 0 4.65% 0</p><p>9月21日 3.85% 0 4.65% 0</p><p>8月20日 3.85% 0 4.65% 0</p><p>7月20日 3.85% 0 4.65% 0</p><p>6月22日 3.85% 0 4.65% 0</p><p>5月20日 3.85% 0 4.65% 0</p><p>4月20日 3.85% -20bp 4.65% -10bp</p><p>3月20日 4.05% 0 4.75% 0</p><p>2月20日 4.05% -10bp 4.75% -5bp</p><p>1月20日 4.15% 0 4.80% 0</p><p>2019年12月20日 4.15% 0 4.80% 0</p><p>11月20日 4.15% -5bp 4.80% -5bp</p><p>10月21日 4.20% 0 4.85% 0</p><p>9月20日 4.20% -5bp 4.85% 0</p><p>8月20日 4.25% -6bp 4.85% --</p><p>自2019年8月20日起，全国银行间同业拆借中心于每月20日（遇节假日顺延）公布贷款市场报价利率。LPR报价行按公开市场操作利率（主要指一年期为主的MLF利率）加点形成的方式，向全国银行间同业拆借中心报价。（完）</p>
