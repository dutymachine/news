[中国LPR连17月持平仍与MLF利率锚同步 净息差缩窄银行调价动力不足](https://cn.reuters.com/article/china-sept-cen-lpr-0922-idCNKBS2GI06F)
------

<div><i>2021-09-22T02:29:39Z</i></div><p>路透上海9月22日 - 中国央行周三公布的9月LPR（贷款市场报价利率）报价显示，一年期利率为3.85%，五年期以上品种利率4.65%，为连续第17个月持平。央行上周对9月中期借贷便利（MLF）等量平价续做，利率锚依旧坚守稳势；而在净息差缩窄的背景下，银行调整LPR报价动力依旧不足。</p><p>东方金诚首席宏观分析师王青和高级分析师冯琳谈到，在净息差收窄至历史低位附近背景下，商业银行下调关键贷款利率定价参考基准的动力不足。</p><p>他们并认为，尽管在跨周期调节下，本轮稳增长政策力度相对温和，年内MLF利率下调的可能性不大，但这并不意味着接下来LPR报价也会一直保持不动。为加大对实体经济支持力度，四季度央行有可能再次实施降准；累积效应下，届时一年期LPR报价下调的可能性会显著增大，幅度可能在0.05-0.1个百分点之间。</p><p>此前的路透调查显示，本月中期借贷便利（MLF）续做时操作利率仍稳如磐石，故95%的受访人士预测本月LPR报价亦将按兵不动。</p><p>央行上周三对9月MLF等量平价续做，面对到期规模偏大，央行此举维稳月中缴税、跨季及缓冲地方债发行压力等可谓一举多得，亦表明稳健偏宽政策基调依然未变。</p><p>中国央行2019年8月推出LPR新机制，一年期品种当年内三次下调，累计降幅为16个基点（bp）至4.15%；新增的五年期以上品种2019年8月首次报在4.85%，随后在当年11月下调一次至4.80%。2020年，一年和五年期LPR各下调两次，其中一年期累计下调30个bp，五年期累计下调15bp。</p><p>以下为LPR新机制报价的各期情况：</p><p>日期 一年 五年以上</p><p>利率 变动 利率 变动</p><p>9月22日 3.85% 0 4.65% 0</p><p>8月20日 3.85% 0 4.65% 0</p><p>7月20日 3.85% 0 4.65% 0</p><p>6月21日 3.85% 0 4.65% 0</p><p>5月20日 3.85% 0 4.65% 0</p><p>4月20日 3.85% 0 4.65% 0</p><p>3月22日 3.85% 0 4.65% 0</p><p>2月20日 3.85% 0 4.65% 0</p><p>1月20日 3.85% 0 4.65% 0</p><p>2020年12月21日 3.85% 0 4.65% 0</p><p>11月20日 3.85% 0 4.65% 0</p><p>10月20日 3.85% 0 4.65% 0</p><p>9月21日 3.85% 0 4.65% 0</p><p>8月20日 3.85% 0 4.65% 0</p><p>7月20日 3.85% 0 4.65% 0</p><p>6月22日 3.85% 0 4.65% 0</p><p>5月20日 3.85% 0 4.65% 0</p><p>4月20日 3.85% -20bp 4.65% -10bp</p><p>3月20日 4.05% 0 4.75% 0</p><p>2月20日 4.05% -10bp 4.75% -5bp</p><p>1月20日 4.15% 0 4.80% 0</p><p>2019年12月20日 4.15% 0 4.80% 0</p><p>11月20日 4.15% -5bp 4.80% -5bp</p><p>10月21日 4.20% 0 4.85% 0</p><p>9月20日 4.20% -5bp 4.85% 0</p><p>8月20日 4.25% -6bp 4.85% --</p><p>自2019年8月20日起，全国银行间同业拆借中心于每月20日（遇节假日顺延）9时30分公布贷款市场报价利率。LPR报价行按公开市场操作利率（主要指一年期为主的MLF利率）加点形成的方式，向全国银行间同业拆借中心报价。（完）</p><p>发稿 边竞；审校 张喜良</p>
