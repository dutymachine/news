[拜登国情咨文：反对中国不公平经贸行为](https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/lu1-03082024115755.html)
------

<p>美国总统拜登本周四发表年度国情咨文演说，并以经贸和印太区域的和平稳定为主轴论述了当前的美中关系。</p><ul><li><span class="result-title"><a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/3-03082024102057.html"><strong>拜登国情咨文首提台海安全　盼与中国竞争而非冲突</strong></a></span></li><li><span class="result-title"><strong> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/9-02282024152355.html">拜登将签署行政令　阻止中俄获得美国个人数据</a> </strong></span></li><li><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/10-12232023172324.html"><strong>拜登签署《国防授权法》：包含抗衡中国及支持台湾</strong></a> </span></li></ul><p>美东时间3月7日晚上9点，美国总统拜登在国会发表一年一度的国情咨文演说。 在讲述与中国的关系时，拜登表示，近年来有很多人认为中国正在崛起而美国正在衰弱，但事实证明，美国依旧拥有全世界最好的经济。</p><p>拜登：“我上任以来，不仅美国国内生产总值增加，对中国贸易逆差也降至十多年来最低点。我们坚决反对中国不公平的经贸行为。”</p><blockquote class="twitter-tweet"><p dir="ltr" lang="zh">【拜登发表国情咨文】<br/>【希望与中国竞争 而不是冲突】<br/>美国总统拜登7日发表国情咨文，他说：“美国正在崛起！”并重申：“美国希望与中国竞争，不是发生冲突。”他还说：“我们坚决反对中国不公平的经济行为，我们坚决维护台海和平稳定。” <a href="https://twitter.com/hashtag/%E6%8B%9C%E7%99%BB?src=hash&amp;ref_src=twsrc%5Etfw">#拜登</a> <a href="https://twitter.com/hashtag/%E5%9B%BD%E6%83%85%E5%92%A8%E6%96%87?src=hash&amp;ref_src=twsrc%5Etfw">#国情咨文</a> <a href="https://t.co/8rESiuPhwI">pic.twitter.com/8rESiuPhwI</a></p>— 自由亚洲电台 (@RFA_Chinese) <a href="https://twitter.com/RFA_Chinese/status/1765985216978731313?ref_src=twsrc%5Etfw">March 8, 2024</a></blockquote><p></p><p>彭博社根据美国商务部数据制作的图表显示，2023年美中贸易逆差降至2790多亿美元，是2010年来的最低点。 如果按占GDP的比例计算，美国对华货物贸易逆差仅为1%，更是2002年以来的最低水平。</p><p>美国拜登政府上任以来，除了延续前任特朗普执政时期的关税措施，还加强了限制对中国人工智能等高科技芯片的出口政策，以阻止美国科技被解放军所用。拜登在周四的演说中强调，美中关系是竞争而非冲突：“在与中国或其他国家的关系中，我们处于更有利的地位，可以赢得 21 世纪的竞争。”</p><p>在台海问题上，拜登重申致力于和平及稳定，并重振美国与印度、日本、韩国、澳大利亚等太平洋地区的联盟及伙伴关系。</p><p>拜登发表国情咨文之际，正值《台湾关系法》签署45周年。 这部法律要求美国政府“提供防御性武器给台湾人民”，并“维持美国的能力，以抵抗任何诉诸武力或使用其他方式高压手段，而危及台湾人民安全及社会经济制度的行动”。</p><p>记者：乔琴恩    责编：何平    网编：瑞哲<span class="result-title"></span></p>
