[中俄"上不封顶"伙伴关系二十大后会改变吗？](https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/cm-10252022121116.html)
------

<p>中国对俄乌战争的立场，外界是雾里看花。中共二十大后的新布局，是否会影响中俄关系以及俄乌战争走向？前美国驻乌克兰大使认为，俄罗斯战败后，中俄伙伴关系也将随之结束，但有美国前官员说，即使如此，中国仍会将俄罗斯视为重要伙伴。</p><p><span class="discreet"><span><span class="searchresultdate"></span></span></span></p><ul><li><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/3-10242022100047.html"><strong>中国9月出口欧美萎缩 与俄贸易增长强劲</strong></a> </span> <span class="discreet"> <span> <span class="searchresultdate"> </span></span></span></li><li><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/8-10132022163734.html"><strong>抗衡中俄 欧盟建议给予波斯尼亚候选国地位</strong></a> </span> <span class="discreet"> <span> <span class="searchresultdate"> </span></span></span></li><li><span class="discreet"><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/lf-10122022134055.html"><strong>共同对抗中俄威胁 加日将共享军事情报</strong></a> </span> <span> <span class="searchresultdate"> </span></span></span></li><li><span class="discreet"><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/6-10122022152727.html"><strong>日澳经济部长对话 对抗中俄经济施压</strong></a> <span> <span class="searchresultdate"> </span></span></span></span></li></ul><p><span class="discreet"><span><span class="searchresultdate"></span></span></span></p><p><span style="font-weight: 400;">自俄罗斯2月24日挥军入侵乌克兰以来，中国至今仍保持“中立”态度，虽然称俄罗斯是战略伙伴关系，但也没有公开支持其侵略乌克兰的行为。</span></p><p><strong>前美国驻乌大使：中俄关系将会消逝</strong></p><p><span style="font-weight: 400;">前美国驻乌克兰大使、目前任职于华盛顿智库大西洋理事会（Atlantic Council）的约翰·赫布斯特（John E. Herbst）认为，中俄两国的密切伙伴关系是出自于对美国和国际秩序的共同敌意，恐怕会有消失的一天。</span></p><p><span style="font-weight: 400;">“驱动这段不寻常、但并非史无前例的中俄关系，是来自（中俄）对美国、以及对我们（美国）与盟友建立的国际秩序，所产生的共同敌意。这掩盖了中俄之间更根本的长期竞争，甚至是彼此之间的仇恨。”赫布斯特在大西洋理事会周二举办的线上研讨会上这么说。</span></p><p><span style="font-weight: 400;">10月12日，联大通过决议谴责俄罗斯非法吞并乌克兰领土的企图，中国选择弃权。中国常驻联合国代表张军上个月曾就俄罗斯在乌克兰占领区举办的入俄公投说，各国主权、领土完整都应该得到尊重。赫布斯特以俄罗斯曾经侵占中国领土为例表示，中俄双方存有基本的差异，而两国近年的密切合作，只是出于俄罗斯总统普京对于乌克兰的“执迷不悟”。</span></p><p><span style="font-weight: 400;">他说：“由于对乌克兰的侵略失败，我们正在看到普京政权开始终结，当普京政权垮台时，我猜想俄罗斯会重新回顾历史，而莫斯科和北京的密切关系将会消失在历史的垃圾箱中，但这可能需要几十年的时间。”</span></p><p><strong>美国前情报官员：俄罗斯可协助中国抗美</strong></p><p><span style="font-weight: 400;">不过，大西洋理事会的中国问题专家、美国前情报官员舒尔曼（David Shullman）却不这么认为。他在研讨会上提到两国共同发表声明称“中俄关系上不封顶”，认为没有理由相信中国将俄罗斯作为其最重要战略伙伴的基本方针已经改变。在今年二月普京发动战争前，中俄两国才共同发表声明称“中俄关系上不封顶”，在九月也曾表示两国将深化军事合作，引起国际担忧。</span></p><p><span style="font-weight: 400;">“与俄罗斯结盟的重要性只会被提高。”舒尔曼说，“中国对于（欧美）盟军对俄罗斯入侵的抵抗感到惊讶，并且想知道未来若发生台海危机时（中国）可能受到的影响。这些都让习近平更强调俄罗斯伙伴关系的价值，即使在俄罗斯逐渐衰落的状态下也是如此。”</span></p><p><span style="font-weight: 400;">舒尔曼分析，中共二十大新布局更集中在中国国家领导人习近平的个人权力，以及他所领导的中国共产党的稳固。因此在中国看来，以美国为首的国际社会正在从经济、国防等各层面“试图威胁”，间接加深了中国与俄罗斯结盟的重要性。</span></p><p><span style="font-weight: 400;">舒尔曼分析：“俄罗斯虽然在衰落，但仍拥有世界上数量最多的核武器、是联合国安理会成员、拥有丰富的能源、粮食和小麦，其领土也与中国接壤 4000多 公里。这一切使俄罗斯成为一个关键，也是唯一可以帮助对抗和分散美国注意力的伙伴，使美国国防计划更加复杂化、确保中国在某些程度上能自给自足，以及减少中国脆弱性。”</span></p><p><strong>欧洲经济依赖中国 恐对中国侵台保持沉默</strong></p><p><span style="font-weight: 400;">上周在比利时布鲁塞尔举行的欧盟峰会上，欧盟表示中国正在全球加强影响力，但也警告中国如果和俄罗斯走近，会影响中国和欧洲的关系。欧洲理事会（ European Council）主席米歇尔（Charles Michel）会后表示，欧盟领袖认定中国构成威胁，但会避免出现直接对抗。</span></p><p><span style="font-weight: 400;">捷克智库“欧洲价值安全政策中心”（European Values Center for Security Policy）主任扬达（Jakub Janda）就说，欧洲国家在经济上普遍依赖中国，因此对中国的野心可能保持沉默，若中国决定效仿俄罗斯而入侵台湾，“欧洲国家恐怕什么也不做”。</span></p><p><span style="font-weight: 400;">“在经济上有很好的联系，这对欧洲人来说很有意义，但如果依赖于中共的意志，那就太糟糕了。这基本上就是我们目前所处的状态，欧洲国家的大部分决策，都是仰赖中国。”扬达说。</span></p><p><span style="font-weight: 400;">他提到乌克兰国会友台小组原定全组访问台湾，却因为中国阻挠仅剩一位议员访台的消息，描述中国能够轻易通过政治和经济影响对欧洲国家施压。扬达认为，欧洲应加速和亚洲国家发展战略伙伴关系，譬如在政治和经济上，与台湾和日本建立联系，以摆脱对中国的依赖，并以此来制衡，使欧洲国家在决策上能够真正独立自主。</span></p><p><span class="discreet"><span><span
