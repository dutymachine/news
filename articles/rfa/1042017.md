[郭飞雄案移送法院起诉 当局意图以创建网站入罪](https://www.rfa.org/mandarin/yataibaodao/renquanfazhi/gf-07222022072744.html)
------

<p>近日，中国异议人士郭飞雄被控涉嫌“煽动颠覆国家政权”案已从检察院起诉至法院，进入法官审理程序。据了解，当局还指控郭飞雄创建以宪政和民主为主题的网站。有知情人士透露，郭飞雄原本有机会离开中国，但由于拒绝妥协才遭当局起诉。</p><p><span class="discreet"><span><span class="searchresultdate"></span></span></span></p><ul><li><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/3-07082022113053.html"><strong>异议人士郭飞雄狱中绝食 恐有生命危险</strong></a> </span> <span class="discreet"> <span> <span class="searchresultdate"> </span></span></span></li><li><span class="discreet"><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/1-06202022100833.html"><strong>中国人权捍卫者郭飞雄“煽颠”案已被移送广州检方起诉</strong></a> </span> <span> <span class="searchresultdate"> </span></span></span></li><li><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/yataibaodao/renquanfazhi/gf-06172022105028.html"><strong>郭飞雄绝食半年后获见律师 坚持政见恐遭长期囚禁</strong></a> </span> <span class="discreet"> <span> <span class="searchresultdate"> </span></span></span></li><li><span class="result-title"> <a class="state-published" href="https://www.rfa.org/mandarin/Xinwen/6-05172022144031.html"><strong>郭飞雄遭灌食 律师申请会见被拒</strong></a> </span> <span class="discreet"> <span> <span class="searchresultdate"> </span></span></span></li></ul><p><span class="discreet"><span><span class="searchresultdate"></span></span></span></p><p>郭飞雄的家属在社交平台证实，郭飞雄被控涉嫌“煽动颠覆国家政权”案已在本月初，从检察院移送至广州市中级人民法院。</p><p>家属透过律师了解到，原名杨茂东的郭飞雄被指使用笔名在网络发表文章；当局并指控他创建“世界宪政民主论坛”网站，发表其本人及他人煽颠文章；去年1月底，他被当局阻止出境后接受自由亚洲电台采访也成为罪状。</p><p>熟悉郭飞雄一家的民运人士杨子立认为，这些指控缺乏法理依据，尤其是涉事网站并没有刊载关于中国政治变革的内容。</p><p>杨子立：“这个网站我当然看过，它应该是一个研究宪政民主的学术性、理论性网站。它里面的文章都是世界各国民主转型的经验、评论、历史纪录，基本上没有国内政治变革的文章。这样一些文章要想把他入罪，至少从技术上应该很难，但是当局也许什么都不顾。”</p><p>郭飞雄去年因要求出境到美国探望患癌的妻子张青而失去自由。杨子立透露，好友因为坚守原则而错失了获准出国的机会。</p><p>杨子立：“一方面，当时他不是写了给李克强和（时任公安部部长）赵克志的公开信吗? 写了之后，当局就给了他护照准备放他走。但是在放他走的时候，（当局）又接到公安部的命令，要他从广州到武汉跟公安部来的人去交谈，肯定是要谈一些条件。他就不去谈，不愿意答应某些条件作为妥协，不愿意签署所谓的认罪书或悔过书。另一方面，他被抓以后一直在绝食斗争，绝食是一种比较激烈的抗议方式。对于当局来讲，他是一块难啃的硬骨头。”</p><p>身在广州市第一看守所的郭飞雄自去年12月起一直绝食抗议，仅接受灌食。今年3月，也就是张青病逝后两个月，他曾写信向姐姐杨茂平交待身后事。</p><p><span class="discreet"><span><span class="searchresultdate"></span></span></span></p><p><figure class="image-richtext image-inline captioned" style="width:1044px;"><img alt="杨茂平（图）担心弟弟郭飞雄会因煽颠罪成而遭重判。（杨茂平独家提供，拍摄日期不详）" height="829" src="https://www.rfa.org/mandarin/yataibaodao/renquanfazhi/gf-07222022072744.html/m0722gf.jpg/@@images/34804410-45a5-4489-8783-0916c0dd9c25.jpeg" title="M0722GF.jpg" width="1044"/><figcaption class="image-caption">杨茂平（图）担心弟弟郭飞雄会因煽颠罪成而遭重判。（杨茂平独家提供，拍摄日期不详）</figcaption><small></small><div id="zoomattribute"><a data-caption="杨茂平（图）担心弟弟郭飞雄会因煽颠罪成而遭重判。（杨茂平独家提供，拍摄日期不详）" data-fancybox="" href="https://www.rfa.org/mandarin/yataibaodao/renquanfazhi/gf-07222022072744.html/m0722gf.jpg" id="single_image" title="杨茂平（图）担心弟弟郭飞雄会因煽颠罪成而遭重判。（杨茂平独家提供，拍摄日期不详）"><img src="/++plone++rfa-resources/img/icon-zoom.png"/></a></div></figure></p><p>杨茂平说，广州天气炎热，很担心弟弟能否挺得住。</p><p>杨茂平：“杨茂东（郭飞雄）现在体重还在继续下降，以前是135到150斤这样的体重，现在降到不到100斤。在这么炎热的天气下，（体重）继续下降会更不好。他仍在被动灌食当中。我跟广东的公安说了。我说，‘你们这被动灌食营养不够，你们得保障他的健康。’他们说，（郭飞雄）的身体也不好，肠道可能出现问题。我又找了广东公安局。我说，‘请你们给他做一个腹部的检查，看有没有大事’。”</p><p>原籍湖北的郭飞雄因投身新公民运动和南方街头运动，而为人所认识。他曾经两次入狱，包括因涉及组织街头活动、要求官员公示财产等而被判囚6年，直到2019年8月才刑满出狱。</p><p>杨茂平：“我肯定不乐观了，前几次都被判入狱，当时张青病危，一直到死都没有见过她丈夫。我希望（郭飞雄）能自由（到美国）去照顾他孩子，但我觉得希望非常渺茫。”</p><p>郭飞雄上月与律师会面期间提出多项诉求，包括要求中国全国人大撤销涉嫌危害国家安全类案件不允许律师会见当事人的规定等。</p><p><span class="discreet"><span><span class="searchresultdate"></span></span></span></p><p>记者：高锋    责编：陈美华、何平    网编：瑞哲</p>
