[津巴布韦反对派质疑大选结果](https://www.rfi.fr/cn/%E7%A4%BE%E4%BC%9A/20230828-rfi-%E6%B3%95%E5%B9%BF-%E5%B0%BC%E5%8F%A4%E6%8B%89-%E9%9D%9E%E6%B4%B2-%E6%B4%A5%E5%B7%B4%E5%B8%83%E9%9F%A6%E5%8F%8D%E5%AF%B9%E6%B4%BE%E8%B4%A8%E7%96%91%E5%A4%A7%E9%80%89%E7%BB%93%E6%9E%9C)
------

<div>28/08/2023 - 14:11</div><img src="https://s.rfi.fr/media/display/3925efba-42bf-11ee-9cdc-005056a90321/w:1280/p:16x9/AP23236287268622.jpg"><p><strong></strong></p><div><p>据本台法广（RFI）法文网报道，津巴布韦的反对派领袖现已正式对2023年08月23日周三总统大选的官方结果提出质疑。该国选举委员会在周末期间发布的计票结果为：领导执政党[非洲民族联盟-爱国阵线](ZANU-PF)的现总统总统埃默森·姆南加古瓦（Président Emmerson Mnangagwa ）得票率52.6%，主要反对派领袖得票率44%。</p><p><strong>-- 津巴布韦反对派领袖指责选举欺诈 -</strong></p><p>本台法广（RFI）通讯员克莱尔（Claire Bargelès）于2023年08月27日星期天夜间发自约翰内斯堡（Johannesbourg）的消息说，津巴布韦反对派[公民变革联盟](CCC-Coalition des Citoyens pour le Changement)领袖纳尔逊·查米萨(Nelson Chamisa)当天面对媒体宣布，赢得了本次总统大选。而一天前，80岁高龄的现总统也对外宣布胜选。</p><a href="https://www.rfi.fr/fr/tag/zimbabwe/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/622a2e80-109b-11ea-a662-005056bf7c53/2018-08-03t134936z_303670663_rc16a89c5960_rtrmadp_3_zimbabwe-election_0.webp"sizes=""/><imgfetchpriority="low" src="https://s.rfi.fr/media/display/622a2e80-109b-11ea-a662-005056bf7c53/2018-08-03t134936z_303670663_rc16a89c5960_rtrmadp_3_zimbabwe-election_0.jpg" alt="Image RFI - Afrique - Zimbabwe" srcset="https://s.rfi.fr/media/display/622a2e80-109b-11ea-a662-005056bf7c53/2018-08-03t134936z_303670663_rc16a89c5960_rtrmadp_3_zimbabwe-election_0.jpg"sizes=""width="2480"height="1725"loading="lazy"/></picture><figcaption><span>RFI Image Archive / Zimbabwe : Nelson Chamisa, le leader de l&#039;opposition - la Coalition des citoyens pour le changement (CCC), lors d&#039;une conférence de presse à Harare. (illustration)</span>                <span>REUTERS - Mike Hutchings / RFI Image Archive</span>            </figcaption></figure></a><p>津巴布韦主要反对派领袖查米萨(Nelson Chamisa)指责本次竞选发生了大规模的公然欺诈。他还告诉媒体，反对派已做好了组阁的准备，但没有明确下一步的具体行动。</p><a href="https://www.rfi.fr/fr/tag/emmerson-mnangagwa/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/9d649acc-2306-11ee-8ea9-005056bfb2b6/AP23189445892369.webp"sizes=""/><imgfetchpriority="low" src="https://s.rfi.fr/media/display/9d649acc-2306-11ee-8ea9-005056bfb2b6/AP23189445892369.jpg" alt="Image RFI - Zimbabwe - Emmerson Mnangagwa" srcset="https://s.rfi.fr/media/display/9d649acc-2306-11ee-8ea9-005056bfb2b6/AP23189445892369.jpg"sizes=""width="2480"height="1653"loading="lazy"/></picture><figcaption><span>RFI Image Archive / Zimbabwe : Président Emmerson Mnangagwa est le leader Zanu-PF, le parti au pouvoir depuis l&#039;indépendance. (illustration)</span>                <span>AP - Peter Dejong / RFI Image Archive</span>            </figcaption></figure></a><p>不过，津巴布韦现总统姆南加古瓦（Président Emmerson Mnangagwa）则表示，本次选举展现了成熟的民主。</p><p><strong>-- 津巴布韦在选举中心周围部署水炮 -</strong></p><p>本台法广（RFI）法文报道提到，津巴布韦上星期三的投票，一直延长到第二天08月24日星期四。在该国首都哈拉雷（Harare），发生了严重的选票不足。在投票站前，有恐吓选民的气氛。对此，非洲联盟（UA）观察团负责人、尼日利亚前总统古德勒克·乔纳森（Goodluck Jonathan）的定性为：潜在的“刑事犯罪”（infractions pénales）。</p><a href="https://www.rfi.fr/fr/tag/goodluck-jonathan/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/7be985e8-9c3e-11eb-959e-005056bf87d6/AP20237643052311.webp"sizes=""/><imgfetchpriority="low" src="https://s.rfi.fr/media/display/7be985e8-9c3e-11eb-959e-005056bf87d6/AP20237643052311.jpg" alt="Image RFI - Afrique - Goodluck Jonathan" srcset="https://s.rfi.fr/media/display/7be985e8-9c3e-11eb-959e-005056bf87d6/AP20237643052311.jpg"sizes=""width="2480"height="1653"loading="lazy"/></picture><figcaption><span>RFI Image Archive / Zimbabwe Election 2023 : l&#039;ancien président nigérian Goodluck Jonathan est le responsable de la mission d&#039;observation de l&#039;Union Africaine.
