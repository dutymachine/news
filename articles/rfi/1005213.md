[智利宣布想要国有化锂产业](https://www.rfi.fr/cn/%E7%A4%BE%E4%BC%9A/20230424-rfi-%E6%B3%95%E5%B9%BF-%E5%B0%BC%E5%8F%A4%E6%8B%89-%E6%99%BA%E5%88%A9%E5%AE%A3%E5%B8%83%E6%83%B3%E8%A6%81%E5%9B%BD%E6%9C%89%E5%8C%96%E9%94%82%E4%BA%A7%E4%B8%9A)
------

<div>24/04/2023 - 13:28</div><img src="https://s.rfi.fr/media/display/28840f52-1a96-11ea-8e3e-005056bf87d6/w:1280/p:16x9/salaratacama.jpeg"><p><strong></strong></p><div><p>南美洲国家智利为当今世界的锂资源大国之一。该国总统已于上周晚些时候宣布，想要对锂产业进行国有化。</p><p><strong>-- 智利开发锂目标是经济和环保两不误 -</strong></p><p>据本台法广（RFI）法文网报导，这一消息是智利总统加夫列尔·博里奇（Président Gabriel Boric）于上周四（2023年04月20日）对外宣布的。锂，作为电器所用电池不可缺的一种金属，国际需求不断增加。而智利目前是全球第二大生产国。锂被列入能源转型的关键金属，具有战略地位。圣地亚哥既想要加强智利的经济，也要保护环境。</p><a href="https://www.rfi.fr/fr/am%C3%A9riques/20230421-le-chili-veut-nationaliser-son-lithium"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/cb194190-123c-11ea-b20a-005056bf7c53/lithium-electric-prices_0.webp"sizes=""/><imgsrc="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI - CHILI - Lithium" srcset="https://s.rfi.fr/media/display/cb194190-123c-11ea-b20a-005056bf7c53/lithium-electric-prices_0.jpg"sizes=""loading="lazy"/></picture><figcaption><span>RFI Image Archive / Matières Premières : le Chili veut nationaliser son lithium. Ici, une vue aérienne des bassins d&#039;une mine de lithium dans le désert d&#039;Atacama dans le nord du Chili. (illustration)</span>                <span>©REUTERS - Ivan Alvarado  / RFI Image Archive</span>            </figcaption></figure></a><p>本台法广（RFI）通讯员Naïla Derroisné发自智利首都圣地亚哥（Santiago）的消息说，博里奇总统（Président BORIC）的战略，是要建立一种公私混合模式。国家将参与到锂的所有生产环节。智利将为此设立一家国有公司，产生更多的财富，用于建造学校、医院和警察局。以便让所有的智利人都得以享有更加有尊严的生活。</p><p><strong>-- 智利开发锂现状是破坏环境和饮用水 -</strong></p><p>本台法广（RFI）法文报导提到，智利在一年里，锂的出口量翻了6倍。圣地亚哥方面有雄心，把智利发展成为全球第一大锂生产国，并尝试着去保护好锂矿开采区所处的那片由盐碱盆地组成的沙漠。</p><a href="https://www.rfi.fr/fr/tag/chili/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/f2d49086-d39b-11ec-b3f5-005056a97e36/chile-6472056.webp"sizes=""/><imgsrc="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI - Chili - Atacama - Désert" srcset="https://s.rfi.fr/media/display/f2d49086-d39b-11ec-b3f5-005056a97e36/chile-6472056.jpg"sizes=""loading="lazy"/></picture><figcaption><span>RFI Image Archive / Le désert d&#039;Atacama, au Chili.</span>                <span>© CC0 Pixabay - Sebastian Baszczyj / RFI Image Archive</span>            </figcaption></figure></a><p>同一报导还指出，在智利，锂的开采破坏了自然环境，直接威胁到当地居民的饮用水。智利目前仅开采了北部的一个矿区salar d’Atacama。但智利还有60多处锂矿资源。当局计划通过国有化方式，研究未来开发新矿的可能性。</p><p>（翻译和编辑：<strong>RFI 法广电台 尼古拉</strong>）</p><p><strong>~.
