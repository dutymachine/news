[斯坦福学者谈中共武统台湾：规划阶段已近尾声，行动的可能性在增加](https://www.rfi.fr/cn/%E4%B8%AD%E5%9B%BD/20221119-%E6%96%AF%E5%9D%A6%E7%A6%8F%E5%AD%A6%E8%80%85%E8%B0%88%E4%B8%AD%E5%85%B1%E6%AD%A6%E7%BB%9F%E5%8F%B0%E6%B9%BE-%E8%A7%84%E5%88%92%E9%98%B6%E6%AE%B5%E5%B7%B2%E8%BF%91%E5%B0%BE%E5%A3%B0%EF%BC%8C%E8%A1%8C%E5%8A%A8%E7%9A%84%E5%8F%AF%E8%83%BD%E6%80%A7%E5%9C%A8%E5%A2%9E%E5%8A%A0)

<div>19/11/2022 - 22:21</div><img src="https://s.rfi.fr/media/display/d9aee90e-3934-11ed-9cf5-005056a9a7b9/w:1280/p:16x9/AP22259405098748.jpg"><p><strong>美国斯坦福大学关注和研究台海局势的学者近日参加世界级播客（the World Class podcast）节目，以“说起台湾时，我们需要说些什么”（What We Need Talk When We Talk About Taiwan）为题发表见解，指出：中国武统台湾，长期规划阶段已接近尾声，付诸行动的可能性正在增加。中国完全有机会在未来五年内对台湾使用武力。                </strong></p><div><p>参加讨论的斯坦福学者是：胡佛研究院中国锐实力与政治学者戴蒙德</p><p>(Larry Diamond)、国际研究所中国军事与安全问题专家马斯特罗(Oriana Skylar Mastro)。</p><p>学者指出，2022年10月，习近平第三次担任中共总书记，习近平统治的延伸不仅对中国，而且对台湾都具有重大影响。</p><p>马斯特罗认为，自 1949 年以来，中国领导人一直在计划和思考如何收复台湾。随着现代化能力的增强，力量平衡已经向中国军队倾斜，成本效益计算有利于北京的野心。长期规划阶段现已接近尾声，付诸行动的可能性正在增加。中国完全有机会在未来五年内对台湾使用武力，并威慑阻止美国干涉的能力。</p><p>戴蒙德提醒，中国军事恐吓明显影响了台湾舆论，令人担忧的是，虽然政治精英认识到局势的危险，但台湾公众的民意调查显示，绝大多数公民仍然认为中国不太可能发动攻击或入侵。</p><p>马斯特罗认为，由于地理位置相近，如果中国行动迅速，中国军队很有可能会在美军有时间进入阵地之前占领台湾。美国太平洋部队都不在亚洲，大多数位于夏威夷和加州，还有日本的一些基地和机场。为了能够有效威慑中国，美国需要更强大的前沿部署军事能力，以便能够阻止或阻止中国军队进入台湾。</p><p>戴蒙德认为，为了最大限度地提高安全性，台湾需要找到加强自身防御、抵抗和威慑中国的能力的方法。增加公民参与军事训练也很重要，重点包括武器训练和城市防御战术。戴蒙德建议台湾和美国通过全面改革当前的武器采购系统，以加快武器的生产和交付，以应对国防需要和突发事件。</p><div data-selfpromo-newsletter></div><div data-selfpromo-app></div></div>
