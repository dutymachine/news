[路透独家揭马斯克将在中国造电动车卖进美国 马斯克斥是虚假消息](https://www.rfi.fr/cn/%E4%B8%AD%E5%9B%BD/20221112-%E8%B7%AF%E9%80%8F%E7%8B%AC%E5%AE%B6%E6%8F%AD%E9%A9%AC%E6%96%AF%E5%85%8B%E5%B0%86%E5%9C%A8%E4%B8%AD%E5%9B%BD%E9%80%A0%E7%94%B5%E5%8A%A8%E8%BD%A6%E5%8D%96%E8%BF%9B%E7%BE%8E%E5%9B%BD-%E9%A9%AC%E6%96%AF%E5%85%8B%E6%96%A5%E6%98%AF%E8%99%9A%E5%81%87%E6%B6%88%E6%81%AF)

<div>12/11/2022 - 10:10</div><img src="https://s.rfi.fr/media/display/31d9ee4c-60e2-11ed-a943-005056bfb2b6/w:1280/p:16x9/AP22125103639848.jpg"><p><strong>路透社引用两名知情人士独家消息指，特斯拉正在考虑向美国出口在中国制造的电动汽车，这一逆转将反映出这个美国汽车制造商在设在中国上海工厂和中国消费者的需求放缓。但这个消息遭到特斯拉老板马斯克否认，并斥责是虚假信息。                </strong></p><div><p>据路透社说，马斯克否认考虑向美国出口中国制造的汽车。</p><p>特斯拉公司首席执行官埃隆·马斯克（Elon Musk）在回应路透社关于该公司考虑将其在中国制造的汽车出口到美国的报道时说是“虚假消息”。</p><p>路透社报道引述知情人士称，特斯拉一直在研究其中国供应商生产的零部件是否符合北美当地法规，如果符合，最早可能在明年将中国制造的 Model Y 和 Model 3 运往北美销售。</p><p>其中一位知情人士说，这也可能为马斯克向加拿大出口特斯拉打开一个渠道。</p><p>该报道称，特斯拉没有立即回应置评请求，但马斯克在故事发布后的 Twitter 帖子中回答“错误”，但没有详细说明。</p><p>特斯拉上海超级工厂在今年早些时候进行升级后，每年可生产 110 万辆电动汽车，使其成为特斯拉生产力最高的制造中心。上海工厂生产 Model 3 轿车和 Model Y 跨界车，在中国销售并出口到欧洲、澳大利亚和东南亚等市场。</p><p>券商 CMBI 的数据显示，直到最近，特斯拉一直在销售或运送其在上海生产的每辆汽车以供出口，但10 月份的库存水平出现了有史以来的最大增幅。</p><p>此外，人民币兑美元贬值、中国原材料价格走低以及美国特斯拉和新车价格上涨等因素综合起来，使中国对美出口具有潜在的成本竞争力。</p><p>据路透社说，特斯拉卖回美国的计划如果获得通过，可能会给美国买家带来新的复杂性。根据美国总统拜登签署成为法律的新电动汽车补贴和生产激励计划的条款，每辆汽车可获得的激励措施可能会因车辆是否进口而异。而且它也可能在政治上引起争议。特斯拉被广泛认为是拜登政府的《通胀降低法案》（IRA）的主要受益者之一，该法案为购买电动汽车提供高达 7,500 美元的回扣，作为旨在推动汽车制造商减少对中国依赖的法律的一部分。</p><p>到目前为止，特斯拉的战略一直是在其位于加利福尼亚州弗里蒙特和德克萨斯州奥斯汀的工厂生产在北美销售的特斯拉汽车。加州工厂是特斯拉的第一家工厂，生产 Model S、Model 3 轿车以及 Model X 和 Model Y 跨界车。今年早些时候开业的德克萨斯工厂生产 Model Y，并将生产特斯拉即将推出的 Cyber​​truck。</p><p>特斯拉还在今年早些时候在柏林开设的一家工厂提高了产量。其中一位消息人士称，柏林工厂的产量将减少对中国部分出口产品的需求。</p><p>与此同时，特斯拉汽车在中国和美国销售的价格差距一直在扩大，这既反映了美国价格的上涨，也反映了中国的新折扣。在中国，CMBI 分析师警告称“价格战”即将到来，特斯拉上个月将中国 Model 3 和 Model Y 的起步价下调了 9%。</p><p>特斯拉在中国以 49,344 美元的价格出售 Model Y，而在美国的售价为 65,990 美元。中国制造的汽车面临 27.5% 的美国关税，而轻型卡车面临 25% 的关税。中国是全球最大的汽车市场，对进口汽车征收 15% 的关税。</p><p>据路透社说，特斯拉不会是第一家将中国制造的汽车运往美国的美国汽车制造商。通用汽车向美国进口了别克 Envision SUV，但未能成功申请豁免特朗普政府征收的 25% 的美国关税。</p><div data-selfpromo-newsletter></div><div data-selfpromo-app></div></div>
