[坦桑尼亚空难 非洲数政要致哀](https://www.rfi.fr/cn/%E9%9D%9E%E6%B4%B2/20221107-%E5%9D%A6%E6%A1%91%E5%B0%BC%E4%BA%9A%E7%A9%BA%E9%9A%BE-%E9%9D%9E%E6%B4%B2%E6%95%B0%E6%94%BF%E8%A6%81%E8%87%B4%E5%93%80)

<div>07/11/2022 - 18:53</div><img src="https://s.rfi.fr/media/display/44b1e20e-5e03-11ed-986a-005056bf30b7/w:1280/p:16x9/2022-11-06T130724Z_1412333236_RC2BGX9M7OB8_RTRMADP_3_TANZANIA-ACCIDENT.JPG"><p><strong>在濒临印度洋的东非国家坦桑尼亚（Tanzanie），周日发生一起空难事故。目前已知至少19人遇难。据报，事发时，气象条件恶劣。飞机掉落在非洲最大的淡水湖维多利亚湖（Lac Victoria）里。机身处于水中，距岸边100米。非洲多名政要向遇难者家属表示致哀。北京也表示哀悼，并透露机上有一名中国籍乘客受伤，目前在医院接受治疗。                </strong></p><div><p>据本台法广RFI法文网报导，在东非国家坦桑尼亚的西北地区当地时间2022年11月06日星期天早上发生一起空难事故。现场的图像显示，飞机大部分处于水中，只剩下机身的后半段仍露出水面。许多船只和救援队在飞机周围，试图把飞机从水中拖出。</p><a href="https://www.rfi.fr/fr/afrique/20221106-tanzanie-19-morts-dans-le-crash-d-un-avion-qui-s-est-ab%C3%AEm%C3%A9-dans-le-lac-vitoria"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/77fd3cfc-5e04-11ed-991d-005056bf30b7/2022-11-06T131825Z_476346783_RC2AGX9TWAXY_RTRMADP_3_TANZANIA-ACCIDENT.webp"sizes=""/><imgsrc="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI Archive - Tanzanie - ACCIDENT - Crash" srcset="https://s.rfi.fr/media/display/77fd3cfc-5e04-11ed-991d-005056bf30b7/2022-11-06T131825Z_476346783_RC2AGX9TWAXY_RTRMADP_3_TANZANIA-ACCIDENT.JPG"sizes=""loading="lazy"/></picture><figcaption><span>RFI Image Archive / Tanzanie: 19 morts dans le crash d&#039;un avion qui s&#039;est abîmé dans le lac Victoria, le dimanche 6 novembre 2022. (illustration)</span>                <span>REUTERS - STRINGER / RFI Image Archive</span>            </figcaption></figure></a><p>一名地方官员此前曾透露，遇难飞机上共有43人，其中39名乘客，2名飞行员和2名机组成员。这架飞机所属的精密航空（Precision Air）在一名声明中宣布，有24人得以劫后余生。</p><p><strong>事发时当地气象状况差 -</strong></p><p>这家航班编号为PW494 的客机，当天从坦桑尼亚的海港城市达累斯萨拉姆（Dar es Salaam）起飞，目的地是位于该国西北的布科巴机场（Bukoba）。地方上的警方指挥官表示，事发时，当地有强风暴雨。在坠机事故发生前，有目击证人看到飞机在空中向机场靠近，但飞行方式不稳定。当时的能见度低。</p><p><strong>非洲多名政要表示哀悼 -</strong></p><p>刚刚在上周末结束访华行程回国的坦桑尼亚女总统萨米娅·苏卢胡·哈桑（Samia Suluhu Hassan）呼吁民众保持平静，并向事故受害者致哀。</p><a href="https://www.rfi.fr/fr/tag/tanzanie/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/b6148d00-b03c-11eb-9144-005056a964fe/AP21096574684770.webp"sizes=""/><imgsrc="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI Archive - Tanzanie" srcset="https://s.rfi.fr/media/display/b6148d00-b03c-11eb-9144-005056a964fe/AP21096574684770.jpg"sizes=""loading="lazy"/></picture><figcaption><span>RFI Image Archive / Tanzanie: 19 morts dans le crash d&#039;un avion qui s&#039;est abîmé dans le lac Victoria.
