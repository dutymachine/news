[加蓬首次加入[汉语桥]中文赛](https://www.rfi.fr/cn/%E9%9D%9E%E6%B4%B2/20220618-rfi-%E6%B3%95%E5%B9%BF-nicolas-%E5%B0%BC%E5%8F%A4%E6%8B%89-%E9%9D%9E%E6%B4%B2-%E5%8A%A0%E8%93%AC%E9%A6%96%E6%AC%A1%E5%8A%A0%E5%85%A5-%E6%B1%89%E8%AF%AD%E6%A1%A5-%E4%B8%AD%E6%96%87%E8%B5%9B)

<div>18/06/2022 - 23:53</div><img src="https://s.rfi.fr/media/display/d04fc080-0ff3-11ea-b804-005056a99247/w:1280/p:16x9/carte-gabon_1_0.gif"><p><strong>[汉语桥]（pont vers le chinois / Chinese Bridge）世界大学生中文比赛始于2002年，现已进入第21个年头。在西非，濒临大西洋几内亚湾的加蓬今年（2022年）首次参加第21届[汉语桥]（pont vers le chinois / Chinese Bridge）中文竞赛。本台法广RFI在当地的通讯员于本周现场见证了这一赛事，并发回了部分实地采访录音片段。                </strong></p><div ><p>据本台法广RFI法文网2022年06月18日星期六报导，中国今年主办第21届[汉语桥]（pont vers le chinois / Chinese Bridge）世界大学生中文比赛。</p><a href="https://www.rfi.fr/fr/tag/gabon/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/0ddcece8-e370-11ea-8238-005056a98db9/gettyimages-528106674_0.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI Archive - Gabon - Libreville" srcset="https://s.rfi.fr/media/display/0ddcece8-e370-11ea-8238-005056a98db9/gettyimages-528106674_0.jpg"sizes=""loading="lazy"/></picture><figcaption><span >RFI Image Archive / Gabon : la capitale est Libreville. Ici, une vue de Libreville au Gabon (illustration)存档图片 / 加蓬 : 首都利伯维尔</span>                <span>Getty Images / RFI Image Archive</span>            </figcaption></figure></a><p> </p><p>加蓬首次参加这一全球学习汉语大学生的盛会。利伯维尔孔子学院（Institut Confucius）的学子们于06月17日周五各展其长，争夺桂冠。</p><p><strong>参赛大学生们各有千秋 -</strong></p><p>本台法广RFI通讯员伊芙-罗兰（Yves-Laurent Goma）发自利伯维尔（<strong>Libreville</strong>也被译作自由城)的消息描述说，15位候选人逐一面对评委会。候选大学生克罗德（Claude）刚刚所作的自我介绍赢得了其同伴们的热烈掌声。他还用流利的汉语说：“学习汉语，这并不难”。</p><div><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/ac141eec-15f6-11ea-b3bc-005056a99247/mada_01_0.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI Archive - Afrique - Confucius" srcset="https://s.rfi.fr/media/display/ac141eec-15f6-11ea-b3bc-005056a99247/mada_01_0.jpg"sizes=""loading="lazy"/></picture><figcaption><span >RFI Image Archive / Afrique : la Chine organise cette année (2022) la 21e édition du concours baptisé « pont vers le chinois ». (illustration)</span>                <span>RFI Image / Alice Milot</span>            </figcaption></figure></div><p>最终摘取桂冠的优胜者是一位名叫萨巴斯（Sabasse）的大学生。他对赢得大奖感到十分高兴。他表示，这一比赛是机会面前人人平等。</p><p><strong>参赛大学生们作风正派 -</strong></p><p>本台法广RFI通讯员指出，面对竞争，候选学子们都遵规守矩。他们的动力不在于竞赛，而是对学习汉语的热情。他们认为，这是一种未来的语言。</p><p><strong>加蓬一名大学生的梦想 -</strong></p><p>参赛大学生克罗德（Claude）在接受本台法广RFI通讯员采访时说，在家中，他时常与父母辩论。家人们认为他是一名亲共产党人。而他认为，中国将在两年内领导世界。克罗德（Claude）的梦想是到国外进修中文的硕士学位。</p><p><strong>加蓬中文教育发展迅速 -</strong></p><p>加蓬首都孔子学院创立于2018年，如今有千名大学生在此学习汉语。杰雷米（Jérémie）是这里的一名学长。他鼓励其他学子们学习汉语。他认为，这是一种战略性语言。</p><div><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/3eb09214-1078-11ea-8bf8-005056a99247/Confucius_Tang_Dynasty.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI Archive - Confucius" srcset="https://s.rfi.fr/media/display/3eb09214-1078-11ea-8bf8-005056a99247/Confucius_Tang_Dynasty.jpg"sizes=""loading="lazy"/></picture><figcaption><span >RFI Image Archive / La Chine organise cette année (2022) la 21e édition du concours baptisé « pont vers le chinois ».
