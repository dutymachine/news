[马达加斯加:力拓面对社区风潮](https://www.rfi.fr/cn/%E9%9D%9E%E6%B4%B2/20220522-rfi-nicolas-%E6%B3%95%E5%B9%BF-%E5%B0%BC%E5%8F%A4%E6%8B%89-%E5%8D%B0%E5%A4%AA-%E9%A9%AC%E8%BE%BE%E5%8A%A0%E6%96%AF%E5%8A%A0-%E5%8A%9B%E6%8B%93%E9%9D%A2%E5%AF%B9%E7%A4%BE%E5%8C%BA%E9%A3%8E%E6%BD%AE)

(illustration)</span>                <span>PFFD - CC - Wikimedia Commons / RFI Image Archive</span>            </figcaption></figure></a><p>鉴于当地在本周发生争执后，示威者自周四以来封锁送往矿场的唯一通道，企业因此被迫宣布停产。企业管理层与示威者领袖之间无法达成协议，双方处于僵持之中。</p><p><strong>双方分歧之一涉及赔偿 -</strong></p><p>综合本台法广RFI通讯员罗尔（Laure Verneau）在过去不到24小时里发自马达加斯加首都安塔那那利佛（Antananarivo）的两则最新消息，参加谈判的一名多凡堡（Fort-Dauphin）居民、当地知名协会的会长透露，目前仍无协议，双方的分歧点在于示威者要求用现金支付赔偿，而企业方面却不想直接给钱。</p><p><strong>政府多名部长参加谈判 -</strong></p><p>本台法广RFI法文网报道说，周六晚，通往QMM矿场的唯一道路仍然被示威者封锁。午夜时，谈判仍在继续。除了矿企领导和当地居民之外，与会的还有水利部长、环境部长和国防部长。</p><p><strong>民众指责矿企污染河流 -</strong></p><p> </p><a href="https://www.rfi.fr/fr/afrique/20220522-madagascar-situation-toujours-tendue-entre-l-entreprise-qmm-et-la-population-locale"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/cb8ca08a-0f45-11ea-b459-005056a9aa4d/148534587_0.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI Archive - Madagascar - Fort Dauphin - rizière" srcset="https://s.rfi.fr/media/display/cb8ca08a-0f45-11ea-b459-005056a9aa4d/148534587_0.jpg"sizes=""loading="lazy"/></picture><figcaption><span >RFI Image Archive / Madagascar : les riverains de Fort-Dauphin accusent QMM, entre autres, d’avoir pollué l’estuaire, mi-février, avec le bassin d’eaux usées qui s’y serait déversé. (illustration)</span>                <span>Getty Images - Olivier Cirendini / RFI Image Archive</span>            </figcaption></figure></a><p> </p><p>当地民众指责合资矿企QMM在二月份中旬污染了河流的水源，要求对水源遭污染，和渔民无法捕捞的损失予以补偿。而企业驳斥上述指责。双方矛盾在周四晚开始激化。示威者一度扣押了27名没有武装的军人作为人质，直到周五下午，政府谈判代表团抵达后才予以释放。</p><p><strong>力拓分企依法享有税惠 -</strong></p><p>本台法广RFI法文网报道指出，跨国矿企力拓集团（Rio Tinto）在马达加斯加依照矿业大投资法（LGIM-Loi sur les Grands Investissements Minier）享有多项税务优惠措施。这家合资企业应该在今年年底就其税务重新展开谈判。</p><p><strong>企业与民众矛盾呈恶化 -</strong></p><p>本台法广RFI法文网在2022年05月21日周六刊出的另一则报导提到，力拓集团（Rio Tinto）旗下马达加斯加分公司QMM与当地民众之间的矛盾已经历时数月之久，但事态不断恶化。矿场现已停产，只能从事维修的工作。企业告知媒体，停产是由于当前的社区动乱，危及矿场的生产安全和职员的人身安全。</p><p>.（翻译编辑：<strong>RFI法广电台 尼古拉</strong>）.</p><p>.</p><div data-selfpromo-newsletter></div><div data-selfpromo-app></div></div>
