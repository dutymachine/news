[几内亚:Kouroussa采金矿工冒险](https://www.rfi.fr/cn/%E9%9D%9E%E6%B4%B2/20220320-%E5%87%A0%E5%86%85%E4%BA%9A-kouroussa%E9%87%87%E9%87%91%E7%9F%BF%E5%B7%A5%E5%86%92%E9%99%A9)
------

<div>21/03/2022 - 00:42</div><img src="https://s.rfi.fr/media/display/c534ebfc-a076-11eb-bc38-005056a964fe/w:1280/p:16x9/carte-kouroussa-guin%C3%A9e.png"><p><strong>西非国家几内亚（Guinée），不仅有大量的铝土矿，黄金储量也极其丰富。过去几周，该国多个金矿接连发生致命意外。本月初，有十多人在一次坑道坍塌事故中遇难。本台RFI特派记者来到几内亚知名的采金地库鲁萨（Kouroussa）做实地采访，并自现场发回报道。                </strong></p><div ><p>本台RFI法文网2022年03月20日星期天报道说，几内亚的多个金矿在过去几周接连发生意外。03月01日，在北部博凯地区（Boké）的Kounsitel, 一条坑道坍塌造成十多人死亡。</p><div><figure><div><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/025310d8-cdcf-11eb-9749-005056bfd1d9/kounsitel-%20Gaoual-Guin%C3%A9e.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="kounsitel- Gaoual-Guinée" srcset="https://s.rfi.fr/media/display/025310d8-cdcf-11eb-9749-005056bfd1d9/kounsitel-%20Gaoual-Guin%C3%A9e.jpg"sizes=""loading="lazy"/></picture></div><figcaption><span >RFI Image / Guinée : la zone de Kounsitel, dans le nord du pays. (illustration).存档图片 / 几内亚北部的Kounsitel</span>                <span>© RFI FULFULDE</span>            </figcaption></figure></div><p>本台RFI法广特派记者得以探访更东面库鲁萨（Kouroussa）附近的一处手工采金矿区。当地的工人们冒着所有风险把稀贵的材料带上地面。</p><p><strong>淘金工作太艰难 - </strong></p><p>据本台RFI特派记者马蒂亚斯（Matthias Raynal）发自几内亚库鲁萨（Kouroussa）的描述，那里的地面如同一块巨大的格鲁耶尔奶酪（gruyère），有孔洞。男人们涌入拉起装满了土地的篮子，并在之后使用金属探测器。</p><a href="https://www.rfi.fr/fr/afrique/20220319-guin%C3%A9e-pr%C3%A8s-de-kouroussa-les-mineurs-prennent-tous-les-risques-pour-de-l-or"><figure><div><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/b80839dc-a7ca-11ec-b21e-005056a97e36/Lanfia%20Kourouma%20orpailleur%20pr%C3%A8s%20de%20Kouroussa.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI - Lanfia Kourouma, orpailleur près de Kouroussa" srcset="https://s.rfi.fr/media/display/b80839dc-a7ca-11ec-b21e-005056a97e36/Lanfia%20Kourouma%20orpailleur%20pr%C3%A8s%20de%20Kouroussa.jpeg"sizes=""loading="lazy"/></picture></div><figcaption><span >RFI Image / Guinée: près de Kouroussa, les mineurs prennent tous les risques pour de l&#039;or. Raynal</span>            </figcaption></figure></a><p>受访人兰菲亚（Lanfia Kourouma）是一名淘金者。他说，人们到此是为了赚取黄金。这里有金，但很深。深度超过十米。本台特派记者描述说，这名淘金者的皮肤和衣服到处都覆盖着白色的灰尘。</p><p><strong>淘金工作太危险 - </strong></p><p>受访人兰菲亚（Lanfia Kourouma）表示，当地已有6年到7年没有发生致命事故。最后一起事故的起因是酒精。当时有个洞已经开始变得有些危险。一名男子收工后留在矿区。而且，他醉酒了，看到有点金子，就想要下去。他留在了那里，陪同的人也失去了一条腿。</p><div><figure><div><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/6569da50-124e-11ea-b7dc-005056bf7c53/Dans%20un%20bar%20caf%C3%A9%20noir%20de%20Kouroussa.webp_0.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI - Kouroussa" srcset="https://s.rfi.fr/media/display/6569da50-124e-11ea-b7dc-005056bf7c53/Dans%20un%20bar%20caf%C3%A9%20noir%20de%20Kouroussa.jpg_0.jpg"sizes=""loading="lazy"/></picture></div><figcaption><span >RFI Image / Guinée: près de Kouroussa, les mineurs prennent tous les risques pour de l&#039;or. (illustration)存档图片：在库鲁萨的一间酒吧内</span>                <span>© RFI / Laurent Correau</span>            </figcaption></figure></div><p>受访者向本台特派记者展示了一个深不见底的井口。井壁内的一些穴孔，形成一条简陋的阶梯。淘金者就是如此下到井底。可以看到一条裂缝，那就是一个痕迹。</p><p><strong>淘金工别无选择 -</strong></p><p>本台RFI法文网报道说，受访人兰菲亚（Lanfia Kourouma）以前是一名司机。他想要结束淘金，重新从事之前的工作。他感慨地说，太艰难了，太危险了。但这是他如今赖以生存的最后希望。</p><p>.（翻译编辑：<strong>RFI法广电台 尼古拉</strong>）.</p><p>.</p><div data-selfpromo-newsletter></div><div data-selfpromo-app></div></div>
