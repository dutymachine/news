<div>16/02/2022 - 01:02</div><img src="https://s.rfi.fr/media/display/4a55f516-8ea5-11ec-bb05-005056bf30b7/w:1280/p:16x9/Capture-991.PNG"><p><strong>我们在上期的节目中为您介绍了由拜登政府在2月11日推出的《美国印太战略》，这一备受关注的政策文件。我们将在本期的节目中继续为您介绍这一拜登政府沿袭其前任特朗普政府，就印太地区这一全球地缘政治新热点所推出的全面性战略内容。我们在上期的节目中提及，美国将根据该战略寻求在印太地区追求五大目标：“1. 推进一个自由和开放的印度-太平洋；2. 建立区域内外的联系 ; 3. 推动印太区域繁荣 ; 4. 加强印太安全 ; 5.  建立区域对跨国威胁的复原力。”                </strong></p><div ><p>就该战略的第三目标——<a href="https://www.whitehouse.gov/wp-content/uploads/2022/02/U.S.-Indo-Pacific-Strategy.pdf" target="_blank">“推动印太区域繁荣”，文件表示</a>：“美国普通大众的繁荣与印度-太平洋息息相关。我们将提出一个创新的新框架，使我们的经济为这一时刻做好准备。我们的努力建立在密切经济一体化的坚实基础上。2020年，美国和印太地区的双向贸易总额为1.75万亿美元，它支持了500多万个印太地区的就业机会。2020年，来自美国（对该地区）的外国直接投资总额超过9690亿美元，在过去十年中几乎翻了一番。美国仍然是东盟成员国的头号投资伙伴，其投资额超过东南亚后三个投资伙伴的总和。而且，美国是印太地区的主要服务出口国，这反过来又促进了地区增长。”</p><p>文件指出：“新冠疫情清楚地表明，需要一个促进广泛经济增长的复苏。这需要投资来鼓励创新，加强经济竞争力，创造高薪工作，重建供应链，并为中产阶级家庭扩大经济机会。在这十年里，印太地区的15亿人将加入全球中产阶级。”</p><p>文件介绍称：“与我们的伙伴一起，美国将提出一个印太经济框架--一个21世纪的多边伙伴关系。这个经济框架将帮助我们的经济体利用快速的科技转型，包括在数字经济领域，并适应即将到来的能源和气候转型。美国将与合作伙伴合作，确保太平洋两岸的公民从这些历史性的经济变革中获益，同时深化我们的一体化。”</p><p>文件补充说：“我们将制定符合高劳工和环境标准的新的贸易方式，并将根据开放原则管理我们的数字经济和跨境数据流动，包括通过一个新的数字经济框架。我们将与我们的伙伴合作，推动具有复原力和安全的供应链，使其多样化、开放和可预测，同时消除障碍，提高透明度和信息共享。我们将在去碳化和清洁能源方面进行共同投资，并在亚太经济合作组织（APEC）中努力促进自由、公平和开放的贸易和投资，在我们担任主办该组织会议的东道主年、2023年及以后。”</p><p>文件称：“我们还将加倍致力于帮助印太伙伴缩小该地区的基础设施差距。通过我们与七国集团伙伴的‘重建更好世界’倡议（B3W），我们将为印太地区的新兴经济体配备高标准的基础设施，使他们能够增长和繁荣，同时在太平洋两岸创造良好的就业机会。在此过程中，我们将促进具有复原力和安全的全球电信，重点关注5G供应商的多样化和开放式无线接入网（O-RAN）技术，并寻求一个有良好条件的电信供应市场，以允许新的、值得信赖的供应商。我们还将与那些在制定21世纪经济活动规则方面发挥主导作用的区域经济伙伴并肩合作。我们将一起利用快速的经济转型作为我们各方的共同机会。”</p><p>就该战略的第四目标——“加强印太安全”，文件表示：“75年来，美国一直保持着支持地区和平、安全、稳定和繁荣所必需的强大和持续的防御存在。美国一直是一个坚定的区域盟友，并将在21世纪继续如此。现如今，我们正在扩大这一作用并使之现代化：美国正在加强我们的能力，以捍卫我们的利益，以及阻止侵略和抵制针对美国领土和我们的盟友及伙伴的胁迫。”</p><p>文件称：“‘一体化威慑’（integrated deterrence）将是我们所采用方法的基石。我们将更紧密地整合我们在各作战领域和冲突范围内的努力，以确保美国与我们的盟友和伙伴一起，能够劝阻或击败任何形式或领域的侵略行为。我们将推动加强威慑和反胁迫的举措，如反对改变领土边界或破坏主权国家海上权利的努力。”</p><p>文件称：“我们将重新关注创新，以确保美国军队能够在快速发展的威胁环境中运作，包括太空、网络空间以及关键和新兴科技领域。我们正在开发新的作战概念，建立更有复原力的指挥和控制，增加我们联合演习和行动的范围和复杂性，并寻求多样化的部队态势机会，这将加强我们与盟友和合作伙伴一起向前和更灵活地行动的能力。”</p><p>文件补充说：“与我们更广泛的战略方针相一致，我们将优先考虑我们唯一最大的不对称力量：即我们的安全联盟和伙伴关系网络。在整个地区，美国将与盟友和伙伴合作，深化我们的互操作性，开发和部署先进的作战能力，因为我们支持这些盟友和伙伴捍卫他们的公民和主权利益。我们将继续使我们与澳大利亚、日本、韩国、菲律宾和泰国的条约联盟现代化；稳步推进我们与印度的主要防务伙伴关系，并支持其作为一个‘净安全提供者’（net security provider）的作用；并建立南亚和东南亚以及太平洋岛国伙伴的防御能力。”</p><p>文件称：“我们还将与区域内外的伙伴合作，维护台湾海峡的和平与稳定，包括支持台湾的自卫能力，以确保有一个根据台湾人民的愿望和最佳利益和平决定台湾未来的环境。在这样做的时候，我们的做法仍然符合我们的‘一个中国’政策和我们在《台湾关系法》、三个联合公报和六项保证中的长期承诺。”</p><p>文件强调：“我们将促进我们在印太地区和其他地区的盟友和伙伴之间的安全联系，包括通过寻找新的机会将我们的国防工业基地联系起来，整合我们的国防供应链，并共同生产关键科技，以加强我们的集体军事优势。在此过程中，我们将以新的方式将我们的印太地区和欧洲伙伴聚集在一起，包括通过澳英美三边安全协议，（即奥库斯协议、AUKUS）伙伴关系。”</p><p>针对朝鲜半岛安全问题，文件表示：“随着朝鲜继续发展破坏稳定的核和导弹计划，我们将继续寻求认真和持续的对话，目标是实现朝鲜半岛的完全无核化，解决其持续的侵犯人权行为，改善朝鲜人民的生活和生计。同时，我们正在加强延伸威慑和与韩国和日本的协调，以应对朝鲜的挑衅，继续准备遏制并在必要时击败对美国和我们盟友的任何侵略，同时加强整个地区的反核扩散努力。在加强对核和弹道导弹系统以及其他新出现的对战略稳定的威胁的延伸威慑的同时，美国将寻求与包括我们的竞争对手在内的一系列广泛的行为者合作，以预防和管理危机。”</p><p>文件谈到：“我们还将进行创新以应对民用安全挑战，扩大美国海岸警卫队的存在、培训和咨询，以加强我们合作伙伴的能力。我们将合作解决和预防恐怖主义和暴力极端主义，包括识别和监测前往该地区的外国战斗人员，制定减轻网上激进主义的方案，并鼓励印太地区的反恐合作。我们将加强区域集体能力，以准备和应对环境和自然灾害；自然、意外或蓄意的生物威胁；以及武器、毒品和人口贩运。我们将改善该地区的网络安全，包括提高我们的合作伙伴保护、恢复和应对网络安全事件的能力。”</p><p>就该战略的第五目标——“建立区域对跨国威胁的复原力”，文件表示：“印太地区是气候危机的中心，但它也是气候解决方案的关键。实现《巴黎协定》的目标将需要该地区的主要经济体将其目标与《协定》的温度目标相一致。这包括敦促中国承诺并实施符合将升温限制在1.5摄氏度所需目标水平的行动。在印太地区，我们共同应对气候危机既是政治需要，也是经济机会，因为世界上70%的自然灾害发生在这里。美国将与合作伙伴一起制定符合将全球温度上升限制在1.5摄氏度的2030年和2050年的目标、战略、计划和政策，并将寻求在印太地区向净零未来过渡时成为首选合作伙伴。”</p><p>声明指出：“通过像Clean Edge清洁绿能指数这样的倡议，我们将激励清洁能源技术的投资和部署，寻求推动能源部门的去碳化，并促进与气候相关的基础设施投资。美国将与合作伙伴合作，减少他们对气候变化和环境退化影响的脆弱性，并将支持关键基础设施的复原力和解决能源安全问题。我们还将努力保障该地区广大海洋的健康和可持续利用，包括通过合法使用其资源，加强研究合作，以及促进有益的商业和运输。”</p><p>声明称：“我们将与印太地区合作，帮助结束新冠疫情，建立应对共同威胁的复原力。我们将与合作伙伴密切合作，加强他们的卫生系统，以抵御未来的冲击，推动对全球卫生安全的投资，并扩大区域平台，以预防、检测和应对紧急情况，包括生物威胁。我们还将通过世界卫生组织 (WHO)、七国集团、二十国集团和其他多边论坛来加强准备和反应。我们将与东盟、亚太经合组织、太平洋岛国论坛（PIF）和其他组织密切协调，推进我们的抗灾工作。”</p><p>针对拜登政府将如何实现这些宏大的全面性目标，该战略文件在其第三大部分——“印太行动计划”的内容中对此加以了概述。</p><p><strong>推动新的资源进入印太地区：</strong></p><p>“建设共享能力需要美国进行新的区域投资。我们将开设新的大使馆和领事馆，特别是在东南亚和太平洋岛国，并增加我们在现有大使馆和领事馆的力量，加强我们的气候、卫生、安全和发展工作。我们将扩大美国海岸警卫队在东南亚和南亚以及太平洋岛国的存在和合作，重点是提供咨询、培训、部署和能力建设。我们将重新把安全援助的重点放在印太地区，包括建立海洋能力和海洋领域意识。我们还将扩大人文交流的作用，包括和平队（Peace Corps）。在美国政府内部，我们将确保我们有必要的能力和专业知识来应对该地区的挑战。自始至终，我们将与国会合作，确保我们的政策和资源得到两党的必要支持，以支持我们强大和稳定的区域作用。”</p><p><strong>领导一个印太经济框架</strong>：</p><p>“我们将在2022年初启动一个新的伙伴关系，促进和推动高标准的贸易，管理数字经济，提高供应链的复原力和安全性，促进对透明、高标准的基础设施的投资，并建立数字连接——加倍加强我们与该地区的经济联系，同时为广泛共享印太机遇作出贡献。”</p><p><strong>强化威慑：</strong></p><p>“美国将捍卫我们的利益，阻止对我们自己国家以及我们的盟友和伙伴的军事侵略——包括台湾海峡两岸，并通过发展新的军事武器、行动概念、军事活动、国防工业倡议以及更具有复原力的部队态势来促进区域安全。我们将与国会合作，为‘太平洋威慑倡议’ (PDI) 和‘海上安全倡议’ (MSI)提供资金。通过三边安全伙伴关系，我们将确定在可实现的最早日期向澳大利亚皇家海军提供核动力潜艇的最佳途径；此外，我们将通过一项关于先进军事力量的具体工作计划，包括网络、人工智能、量子技术和海底能力，深化合作并提高互操作性。”</p><p><strong>加强一个有能力的、统一的东盟：</strong></p><p>“美国正在对美国-东盟关系进行新的投资，包括接待东盟领导人参加历史性的美国-东盟特别峰会，有史以来第一次在华盛顿特区举行。我们将实施超过1亿美元的美国-东盟新倡议。我们还将在整个东南亚地区扩大双边合作，优先努力加强卫生安全，应对海上挑战，增加连通性，并深化人文交流。”</p><p><strong>支持印度的持续崛起和区域领导地位：</strong></p><p>“我们将继续（与印度）建立一种战略伙伴关系，在这种关系中，美国和印度共同努力并通过区域集团促进南亚的稳定；在新的领域，如健康、太空和网络空间进行合作；深化我们的经济和科技合作；并为一个自由和开放的印度-太平洋作出贡献。我们认识到，印度是南亚和印度洋的一个志同道合的伙伴和领导者，活跃在东南亚并与之相连，是四方安全对话和其他区域论坛的推动力，也是区域增长和发展的引擎。”</p><p><strong>落实四方安全对话机制（Quad）：</strong></p><p>“我们将加强四方安全对话作为首要区域集团的地位，并确保其在对印太地区至关重要的问题上取得成果。四方安全对话将在新冠疫情应对措施和全球卫生安全方面发挥区域领导作用，兑现其为该地区和世界提供额外10亿支疫苗的投资。它将推进关键科技和新兴技术方面的工作，推动供应链合作，联合科技部署，并推进共同科技原则。四方安全对话将建立一个绿色航运网络，并将协调卫星数据的共享，以提高海洋领域的意识和气候反应。其成员将合作在南亚、东南亚和太平洋岛国提供高标准的基础设施，并将努力提高他们的网络能力。‘四方安全对话奖学金’（Quad Fellowship）将于2022年正式启动，从所有四个国家招募第一批100名学生，从2023年开始在美国攻读科学、技术、工程和数学领域（STEM）的研究生学位。四方安全对话将继续定期举行领导人和部长级别的会议。”</p><p><strong>扩展美日韩合作：</strong></p><p>“几乎每一个重大的印太挑战都需要美国的盟友和伙伴，特别是日本和韩国之间的密切合作。我们将继续通过三边渠道就朝鲜问题进行密切合作。除了安全问题，我们还将在区域发展和基础设施、关键技术和供应链问题以及妇女的领导力和赋权方面进行合作。我们将越来越多地寻求在三边背景下协调我们的区域战略。”</p><p><strong>在合作下构建太平洋岛国的复原力 ：</strong></p><p>“美国将与合作伙伴一起建立一个多边战略集团，支持太平洋岛国作为安全、独立的行为体建设他们的能力和复原力。我们将通过太平洋地区基础设施基金（PRIF）共同建设气候复原力；协调解决太平洋地区的基础设施差距，特别是信息和通信技术方面的差距；促进运输；并合作改善海上安全，以保障渔业，建立海洋领域的意识，并改善培训和咨询。我们还将优先考虑完成与自由联系邦（密克罗尼西亚联邦、马绍尔群岛以及帕劳，三个太平洋岛国）的《自由联系条约》协定。”</p><p><strong>支持良好的治理和问责制 ：</strong></p><p>“我们将通过帮助合作伙伴根除腐败，包括通过对外援助和发展政策、在七国集团和二十国集团的领导地位以及在开放政府伙伴关系中重新发挥作用，支持印太地区政府作出独立政治选择的能力。我们还与政府、民间社会和记者合作，确保他们有能力揭露和减少来自外国干涉和信息操纵的风险。美国将继续支持缅甸的民主，与盟友和伙伴密切合作，向缅甸军方施压，要求其恢复民主，包括通过可信地实施‘（东盟）五点共识’。”</p><p><strong>支持开放的、具有复原力、安全的和值得信赖的科技
