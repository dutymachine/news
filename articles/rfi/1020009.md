[NBA：再因新疆争议而起辩论](https://www.rfi.fr/cn/%E7%BE%8E%E6%B4%B2/20220118-nba-%E5%86%8D%E5%9B%A0%E6%96%B0%E7%96%86%E4%BA%89%E8%AE%AE%E8%80%8C%E8%B5%B7%E8%BE%A9%E8%AE%BA)
------

<div>18/01/2022 - 12:56</div>Array<p><strong>美国NBA加州 Les Golden State Warriors球队今天官方推文批评该俱乐部一名外围投资者关于漠视新疆批评的言论，划清界限称言论不代表该球队的立场。一名球队投资者Chamath Palihapitiya在直播节目中说，“说实话，没有人关心维吾尔人的情况。”Chamath Palihapitiya还说，在关心别人之前，首先关心美国人自己。美国NBA凯尔特人俱乐部球员坎特批评中国，对Palihapitiya的说法表示愤慨。                </strong></p><div ><p>美国NBA金州勇士队周一批评俱乐部执行委员会的一名成员帕里哈皮蒂亚Palihapitiya说他才不关心 "维吾尔人的情况"，而维吾尔这个少数民族群体是舆论批评中国政权人权记录的一个核心内容。</p><p>据法新社今天报道称， Les Golden State Warriors这支位于旧金山的NBA球队在推特上写道，"帕里哈皮蒂亚Palihapitiya作为一个有限级别的投资者，在勇士队中并没有日常的运营职能，他并不代表我们的特许经营权，他的评估也不反映我们俱乐部的立场评估。" Les Golden State Warriors球队拥有非常庞大的亚洲社区球迷影响。</p><p>Chamath Palihapitiya是在他的All-In播客中讨论美中关系时发表了这一评论。在回答他的共同主持人关于拜登政府对新疆维吾尔人的立场的问题时，他回答说：“说实话，没有人关心维吾尔人的情况。”据帕里哈皮蒂亚Palihapitiya说， "我关心的是，如果中国入侵台湾，美国的经济健康状况可能在瞬间发生变化。但如果你问我是否关心另一个国家的一个阶层的一小部分人，在直到我们能够正确地开始照顾自己之前，我不会把他们放在关心我们自己之上。"</p><p>面对批评，帕里哈皮蒂亚后来对他自己的说法做了解释与修正，承认他可能表现得 "缺乏同情心"。"帕里哈皮蒂亚Palihapitiya还在Twitter上明确地说，“我认为人权很重要，无论是在中国、美国还是其他地方。”</p><p>法新社说，距离北京冬季奥运会开幕（2月4日至20日）只有几天时间，中国当局在其人权记录方面仍然面临很多批评，特别是在对待新疆地区的穆斯林维吾尔少数民族方面。一些非政府组织长期以来一直呼吁抵制奥运会，12月，拜登政府宣布不会派官方代表参加奥运会，因为 "新疆正在发生的种族灭绝和反人类罪以及其他侵犯人权行为"。此后其他国家也纷纷效仿。</p><p>在中国，自从两年多前休斯顿火箭队的一名高管对香港的民主运动表示支持后，中国官方的中央电视台就没有转播过NBA比赛。</p><p>而自从波士顿凯尔特人队的球员坎特Enes Kanter Freedom公开批评称中国国家主席习近平为是"残暴的独裁者 "，并谴责中国的西藏政策后，中国的腾讯公司也停止播放波士顿凯尔特人队的比赛。</p><p>据法新社说，在推特上，这位失去土耳其公民身份但入籍美国的凯尔特人俱乐部球员转发了一段帕里哈皮蒂亚的言论视频，坎特评论说：“当种族灭绝发生时，正是这样的人姑息让它发生的。你真无耻!”</p><div data-selfpromo-newsletter></div><div data-selfpromo-app></div></div>
