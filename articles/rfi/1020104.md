[医学:破译非洲基因ADN的挑战](https://www.rfi.fr/cn/%E7%A4%BE%E4%BC%9A/20220116-rfi-%E6%B3%95%E5%B9%BF-%E5%B0%BC%E5%8F%A4%E6%8B%89-reportage-%E5%8C%BB%E5%AD%A6-%E7%A0%B4%E8%AF%91%E9%9D%9E%E6%B4%B2%E5%9F%BA%E5%9B%A0adn%E7%9A%84%E6%8C%91%E6%88%98)
------

<div>16/01/2022 - 03:57</div>Array<p><strong>更好地认识非洲基因（ADN/DNA），可以让人类了解到哪些秘密？就此问题，在欧洲新闻学中心（EJC-Centre européen pour le journalisme）支持下，本台RFI法文网在«国际报导»专栏（Chronique Reportage International）的最新一集节目中，意大利通讯员播出了对在非洲和在美国从事基因工作参与者的部分采访片段。                </strong></p><div ><p>据本台RFI法文网2022年01月15日星期六报导，世界上有越来越多的科学家都提出了这样一个问题，更好地认识非洲基因可以了解到哪些秘密？非洲大陆有着全球最丰富的遗传多样性，然而，在现有基因数据库中，非洲基因的比率还不到2%。本台RFI法文网在«国际报导»专栏（Chronique Reportage International）刊出的这篇报导法文原标题为：Santé: les enjeux du décryptage de l'ADN africain。中文可译作：健康：破译非洲基因DNA的重要性。在此作简介。</p><div><figure><div><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/66d84dfc-356d-11eb-8ba3-005056bf87d6/reportage-international.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="FMM / RFI : Chronique Reportage International" srcset="https://s.rfi.fr/media/display/66d84dfc-356d-11eb-8ba3-005056bf87d6/reportage-international.png"sizes=""loading="lazy"/></picture></div><figcaption><span >FMM / RFI : Chronique Reportage International</span>                <span>© Studio Graphique FMM</span>            </figcaption></figure></div><p>本台RFI意大利通讯员赛西莉（Cécile Debarge）在其发自西西里首府巴勒莫或帕勒摩（Palerme/Palermo）的报导中指出，非洲黑人占世界人口近17%。然而，在科学文献与医学探讨所依赖的基因研究中，仅占受试者的2%。就此，肯尼亚的一位女独立咨询顾问珍妮弗（Jenniffer）指出，缺乏数据对科研人员有直接后果。由于缺乏资金和缺少科学家去收集这些数据，人们无法回答想要得知的所有问题。然而，这对凑齐拼图必不可少。非洲的多样性是辽阔的。如果只有这些数据的一小部分，正如做拼图缺少图块。</p><p><strong>数据库多样化能使药物更有效 -</strong></p><p>珍妮弗（Jenniffer）曾在2017年初，领导过非洲科学院（AAS）的基因部门。她解释说，以一个被命名为PCSK9的基因（GENE）为例，它对调节血液中的脂类必不可少。研究显示它存在一种变异，在非洲人中更常见，而在欧洲人中罕见。这种变异就是在非洲人被纳入研究时才被发现的，并在此后开发了针对血脂的治疗办法。而血脂则是造成梗死的诱因。</p><div><figure><div><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/2883335c-756b-11ec-bde4-005056a90284/dna-163466.webp"sizes=""/><img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" alt="Image RFI Archive -
