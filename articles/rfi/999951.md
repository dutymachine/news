[几内亚赴迪拜寻融资30亿美元](https://www.rfi.fr/cn/%E4%B8%AD%E4%B8%9C/20240215-rfi-%E6%B3%95%E5%B9%BF-%E5%B0%BC%E5%8F%A4%E6%8B%89-%E9%9D%9E%E6%B4%B2-%E5%87%A0%E5%86%85%E4%BA%9A%E8%B5%B4%E8%BF%AA%E6%8B%9C%E5%AF%BB%E8%9E%8D%E8%B5%8430%E4%BA%BF%E7%BE%8E%E5%85%83)

(illustration)</span>                <span>AP - Kamran Jebreili / RFI Image Archive</span>            </figcaption></figure></a><p>本台法广（RFI）非洲通讯员发自科纳克里（Conakry）的消息说，从周四至周五，几内亚政府与资方在迪拜有约。这一会晤之所以被移到海湾地区是为了能够尽可能多的吸引到合作方。计划部下属通讯服务单位就是这么对外解释的。当局携带的行李箱中放着几内亚至2025年前的发展新框架（PRI）。几内亚当局还在一份声明中强调，这是唯一的参考框架。</p><p><strong>-- 几内亚发展方案估算总额上百亿美元 -</strong></p><p>几内亚的这一发展方案含有60来个项目，涉及所有的单位：经济部、基础设施、教育、农业。科纳克里（Conakry）的目标是要在迪拜的会晤中，敲定这个总额120亿美元方案的所需资金。这笔钱的大部分来源将由政府想办法筹集，这部分的比率为70%。该国计划部特别指出，几内亚还缺30亿美元。</p><a href="https://www.rfi.fr/fr/tag/guin%C3%A9e/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/b23b71b4-afc4-11ee-8fff-005056a90284/w1280-p16x9-2023-12-18T112933Z_868861358_RC2NZ4AKZQ6J_RTRMADP_3_GUINEA-OILSTORAGE-BLAST.webp 1200w"sizes=""><imgfetchpriority="low" src="https://s.rfi.fr/media/display/b23b71b4-afc4-11ee-8fff-005056a90284/w1280-p16x9-2023-12-18T112933Z_868861358_RC2NZ4AKZQ6J_RTRMADP_3_GUINEA-OILSTORAGE-BLAST.jpg" alt="存档图片 / 几内亚：卡卢姆（Kaloum）燃料库于2023年12月中下旬发生爆炸。RFI Image Archive / Guinée : l’explosion du dépôt d’hydrocarbures de Kaloum en décembre 2023." srcset="https://s.rfi.fr/media/display/b23b71b4-afc4-11ee-8fff-005056a90284/w1280-p16x9-2023-12-18T112933Z_868861358_RC2NZ4AKZQ6J_RTRMADP_3_GUINEA-OILSTORAGE-BLAST.jpg 1200w"sizes=""width="1280"height="720"loading="lazy"></picture><figcaption><span>存档图片 / 几内亚：卡卢姆（Kaloum）燃料库于2023年12月中下旬发生爆炸。RFI Image Archive / Guinée : l’explosion du dépôt d’hydrocarbures de Kaloum en décembre 2023.</span>                <span>©  Reuters - Stringer / RFI Image Archive</span>            </figcaption></figure></a><p>在卡卢姆（Kaloum）燃料库发生爆炸后，这一灾难如今对几内亚的国家财政造成负担。这一灾难造成严重的毁坏。企业也遭受重创。这一灾难造成持续多日的燃油匮乏。</p><p><strong>-- 世界银行出席几内亚在迪拜的融资会 -</strong></p><p>几内亚计划部表示，周四在迪拜，与会人有政府首脑、部长，以及世界银行这样的国际机构，和几内亚企业。</p><a href="https://www.rfi.fr/fr/afrique/20231221-guin%C3%A9e-l-explosion-du-d%C3%A9p%C3%B4t-d-hydrocarbures-de-conakry-vue-depuis-l-espace"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/25f36d68-a021-11ee-924b-005056bfb2b6/INFOX%20%2862%29.webp 1200w"sizes=""><imgfetchpriority="low" src="https://s.rfi.fr/media/display/25f36d68-a021-11ee-924b-005056bfb2b6/INFOX%20%2862%29.png" alt="存档图片 / 卡卢姆（Kaloum）燃料库爆炸所带来的灾难给几内亚的国家财政造成负担。（2023年12月中下旬）RFI Afrique / Guinée : la catastrophe de Kaloum pèse sur les finances de l’État.
