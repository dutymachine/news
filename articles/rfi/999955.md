[非洲大西洋沿岸国评估渔业资源](https://www.rfi.fr/cn/%E7%A4%BE%E4%BC%9A/20240218-rfi-%E6%B3%95%E5%B9%BF-%E5%B0%BC%E5%8F%A4%E6%8B%89-%E9%9D%9E%E6%B4%B2-%E9%9D%9E%E6%B4%B2%E5%A4%A7%E8%A5%BF%E6%B4%8B%E6%B2%BF%E5%B2%B8%E5%9B%BD%E8%AF%84%E4%BC%B0%E6%B8%94%E4%B8%9A%E8%B5%84%E6%BA%90)
------

<div>18/02/2024 - 18:18</div><img src="https://s.rfi.fr/media/display/6a2525ae-af2f-11ee-8e8e-005056a90284/w:1280/p:16x9/iStock-1489378220.jpg"><p><strong></strong></p><div><p> </p><p><strong>-- 贝宁水域的渔业评估由摩洛哥提供技术支持 -</strong></p><p>据本台法广（RFI）法文网报道，位于大西洋沿岸的多个非洲国家正在应对他们海洋资源所遭受的威胁。面对资源储备崩溃，它们决定展开评估行动，以了解它们水域内的渔业资源状况。在摩洛哥专家的参与下，目前贝宁水域内的这一行动正在展开。</p><a href="https://www.rfi.fr/tw/%E9%97%9C%E9%8D%B5%E8%A9%9E/%E8%B2%9D%E5%AF%A7/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/9d150d48-a768-11eb-85b0-005056bff430/1618328562015.webp 1200w"sizes=""><imgfetchpriority="low" src="https://s.rfi.fr/media/display/9d150d48-a768-11eb-85b0-005056bff430/1618328562015.jpg" alt="存档图片 / 非洲国家贝宁的科托努自治港大门出入口。RFI Image / Bénin : L&#039;entrée du port de Cotonou, poumon de l&#039;économie béninoise. Avril 2021. (illustration)</span>                <span>© RFI/Aurore Lartigue</span>            </figcaption></figure></a><p>本台法广（RFI）通信员让-吕克（Jean-Luc Aplogan）2024年02月17日周六发自科托努（Cotonou）的消息说，在贝宁的科托努自治港（port autonome de Cotonou），这项行动的启动十分隆重。有三个国家的渔业部长到场。这三国部长分别来自贝宁、科特迪瓦和摩洛哥。摩洛哥提供了一艘十分现代化的海洋学船，长50米。船上装备了多个实验室和近15位摩洛哥科研人员。</p><p><strong>-- 科特迪瓦动物与渔业资源部长简介科考任务 -</strong></p><p>[大西洋沿岸非洲国家间渔业合作部长级会议]（ATLAFCO / COMHAFAT）的现任执行主席是科特迪瓦动物与渔业资源部长西迪·蒂耶莫科·杜尔（Sidi Tiémoko TOURÉ）。面对贝宁和摩洛哥两国的部长们，他在仪式现场简介了摩洛哥和贝宁科研人员们的工作任务：绘制不同水域的海底；辨识和登记所有的物种，以便有助于制定措施，长期保护生物多样性和资源持久性。</p><p> </p><a href="https://www.rfi.fr/fr/environnement/20240217-plusieurs-pays-c%C3%B4tiers-africains-m%C3%A8nent-ensemble-une-%C3%A9valuation-de-leurs-r%C3%A9serves-halieutiques"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/f74c0be4-1179-11ea-a838-005056a99247/port-de-Cotonou_432.webp 1200w"sizes=""><imgfetchpriority="low" src="https://s.rfi.fr/media/display/f74c0be4-1179-11ea-a838-005056a99247/port-de-Cotonou_432.jpg" alt="存档图片 / 贝宁的科托努渔港RFI Image Archive / Bénin : le port de pêche de Cotonou. (illustration)</span>                <span>(photo :V.Cagnolari)</span>            </figcaption></figure></a><p>这次的勘探行动将持续15天。在这背后，还必须采取紧急措施，避免海洋生态体系的崩溃。科特迪瓦在2023年实施了休渔。杜尔部长（Sidi Tiémoko TOURÉ）说取得巨大成功。</p><p><strong>-- 大西洋沿岸非洲国家间渔业合作部长级会议 -</strong></p><p>在贝宁之前，摩洛哥的这艘科考船已经通过了利比里亚水域。[大西洋沿岸非洲国家间渔业合作部长级会议]（ATLAFCO / COMHAFAT）这一区域间国际组织有22个成员国。它们准备一起对抗海洋生态体系被毁和物种储备减少。</p><a href="https://www.rfi.fr/fr/tag/oc%C3%A9ans/"><figure><picture><sourcetype="image/webp"srcset="https://s.rfi.fr/media/display/8c7b7c64-121e-11ea-b044-005056a99247/saintehelene_0.webp 1200w"sizes=""><imgfetchpriority="low" src="https://s.rfi.fr/media/display/8c7b7c64-121e-11ea-b044-005056a99247/saintehelene_0.jpg" alt="l&#039;île de Sainte-Hélène, territoire d&#039;Outre-mer du Royaume-Uni, située dans l&#039;Océan Atlantique entre le continent africain et les Amériques.
