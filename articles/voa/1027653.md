[学者苏罕金:中国民营保安业的大发展与问题](https://www.voachinese.com/a/interview-with-sergey-sukhankin-on-china-s-private-security-contractors-02062023/6949688.html)

#999;">资料照 - 北京郊外一座豪宅外的保安人员。照片摄于2010年9月29日。</small></div><p>自1980年代中期以来，随着中国的市场化经济改革的发展，历史悠久又销声匿迹几十年的民营保安服务行业再度出现并在过去几年呈现大发展。但在长期研究中国这一行业的学者谢尔盖·苏罕金看来，尽管中国民营保安服务业公司众多，雇人数百万，该行业专业化水平始终难以提高。而且在可见的将来也难提高。</p><p>谢尔盖·苏罕金（Sergey Sukhankin）是总部设在美国首都华盛顿的国防政策智库詹姆斯敦基金会的高级研究员。该基金会日前发表了他的研究报告，标题是“中国民营保安合同服务业解剖”。</p><p>该研究报告展示了中国民营保安服务业自1980年代中期以来的大发展，指出了中国这一行业在国内国外的发展所拥有的优势和劣势，以及该行业提升其专业性并进一步发展所面临的一些难以克服的障碍。</p><div ><img  src="https://images.weserv.nl?url=gdb.voanews.com/06a20000-0aff-0242-21a0-08db08528be6_w900.jpg" alt="谢尔盖·苏罕金（Sergey Sukhankin）" border="0"/><div><small style="color: #999;">谢尔盖&#183;苏罕金（Sergey Sukhankin）</small></div></div><p>在苏罕金看来，中国民营保安服务业在国内和国际间进一步发展意味着它必须提升其专业性。然而，教育水平、政治制度和文化因素对其专业生的提升构成了阻碍。他在研究报告中指出：</p><p>（中国民营保安服务业的）“一个薄弱领域是专业水平，以及对训练和发展（狭义的）专业技能的不平衡的做法。就像先前指出的那样，中国在民营保安合同商雇人的领域的做法是业余的。从中国现有的五千七百多万退伍军人当中招募民营保安合服务业从业人员的做法从长远来看难以持续。这里的基本问题不是人员的数量问题而是质量和达到当代民营军事/保安合同商的标准的能力问题。就像一些中国的专家指出的那样，中国武装警察和退伍军人可以娴熟地使用武器，但他们 ‘连写一份简单的保安报告都感到困难’。这就意味着他们的文化水平和训练水平极低。</p><p>“就像学者阿尔都伊诺（Arduino）正确指出的那样，“要想在一带一路的线路上获得成功，中国所需要的不是更多的警卫，而是有效率和具有灵活性的经理人员......很多受雇于中国保安合同商的中国退伍军人所展示的那种盲目服从的态度更多的是一种障碍而不是资产。这一局面意味着为了符合当代民营保安合同商行业的规范、标准和要求，中国的合同商必须改组它们整个的命令和控制架构。但考虑到中国的规范和文化，中国合同商做到这一点的能力令人十分怀疑。”</p><p>日前苏罕金接受了美国之音的采访。苏罕金表达的是他的个人观点，不代表美国之音。</p><p><strong>中国民营保安合同商跟俄罗斯瓦格纳集团区别何在</strong></p><p><strong>金哲问：</strong>谈起中国民营的海外海外保安服务业，很多人难免想起俄罗斯的瓦格纳集团（The Wagner Group）。瓦格纳集团跟中国的民营保安合同商有什么区别？国际间是否有人担忧中国的民营保安服务合同商将来会变成中国版的瓦格纳集团？</p><p><strong>苏罕金答</strong>：这是一件需要考虑的非常重要的事情。在我看来，俄罗斯的瓦格纳集团和中国的民营保安服务合同商的区别在于，首先，瓦格纳集团目前所执行的工作任务跟中国民营保安公司所承担的任务和责任完全不同。实际上，瓦格纳集团是直接从事军事行动。</p><p>从瓦格纳集团如今在乌克兰的所作所为来看，它可以被认为是有时候被用来取代俄罗斯武装力量。而中国的民营保安公司就其主要的责任范围来看是保护中国在国外的资产、工人、房屋财产。</p><p>瓦格纳集团跟中国民营保安服务商的第二个重要区别在于它们的构成。瓦格纳集团雇员基本上是雇佣兵，他们当中很多人有从军的经验。今天，这些雇佣兵当中很多人是从俄罗斯监狱里拉来的。但中国的做法完全不同，中国的保安服务公司不会从监狱里招人。</p><p>另外，就保安行业专业性而言，在我看来，瓦格纳集团的专业性要大大高于中国民营保安公司。为什么呢？因为从瓦格纳集团2014年成立时就开始参与一些重大的地区冲突，包括在乌克兰、叙利亚、利比亚的冲突。现在人们可以很难清楚地看到瓦格纳集团的老总普里高津（Yevgeny Prigozhin）明显地展示出有政治抱负。而我认为中国的民营保安服务合同商/公司则完全服从中国政府，他们的老总也没有人谈论，那些人也没有展示任何政治抱负。</p><p>你提的第二个问题是，国际间是否有人担忧中国的民营保安服务合同商将来会变成中国版的瓦格纳集团？这个问题嘛，你知道，天底下什么事情都可能发生，但在我看来，这种事情不太可能发生。</p><p>我们接下来就要说到，中国的政治精英的思维要比俄罗斯的政治精英更为前瞻，更富有战略性。我不认为他们会把权力/力量分配给非政府企业或犯罪人员，让给他们以后有机会挑战政权。因此，我不认为中国的政治精英会准许中国的民营保安合同商或公司演变为瓦格纳集团那样的势力。</p><p><strong>中国民营保安行业的起源与再兴起</strong></p><p><strong>问：</strong>在你的研究报告中，你讲了中国民营保安行业的再兴起。你是否可以跟我们的受众说一下中国的这种行业第一次是什么时候兴起的？</p><p><strong>答</strong>：中国的民营保安合同商的发展可谓源远流长。有些专家把民营保安合同商的原型追溯到宋朝（960-1279），当时所谓的 “镖局文化” 开始兴起。这一趋势在元朝和明朝继续发展，并在清朝达到顶峰。明朝的时候，中国跟外国的商业往来大发展，当时的私营保安行业主要是提供商人武装护送、保障货物和财产安全等服务。</p><p>一些专家指出，中国的民营保安合同商的原型主要是被用来从事保安行动，那些行动很少超出王朝地界范围，而且主要是从事陆地的保安。但有些民营保安合同商的原型也参加反海盗行动。后来，在清朝的经济、政治、军事陷入停止状态的时候，对这些民营保安合同商需求减少了，到了1920年代初期他们基本上消失了。</p><p>在相隔几十年之后，在毛泽东1976年死去之后，随着1984年中国第二阶段市场改革，“镖局文化” 再度兴起。当时中国政府的控制有所放松，引进了一些经济去中心化。尤其是在1984年，在公安局的主管之下，中国首家民营保安合同商在深圳经济特区出现，并为全中国的保安服务公司的形成铺平的道路。</p><p>当然，自那时以来，很多事情发生了变化，我要特别强调的是，当今中国的民营保安业的兴起和发展跟中国的经济增长和对外经济联络和在外经济活动之间的有不可分割的关联。</p><p><strong>中国几百万保安人员做什么</strong></p><p><strong>问：</strong>你在你的研究报告中写道：“现在可以得到的信息显示，在2018年，中国登记在册的民营保安合同商有5000家，雇佣了430万保安人员，但只有20家（雇佣了大约3200人）保安公司提供国际服务。...与此同时，2022年最新的计算显示，中国登记在册的民营保安公司数量大增，增加到7000家。</p><p>为什么中国要雇佣数百万的民营保安公司保安人员？我们知道，中国已经有了一百五十万人的武装警察。这些民营保安公司在中国境内都是做什么呢？</p><p><strong>答：</strong>谢谢你提出这个问题。就像你指出的那样，中共的武警至少一百五十万人。他们所承担的任务跟民营保安合同商是大不一样的。在这里有一个非常重要的一点需要说明。</p><p>我们在提到中国的民营保安服务合同商或公司的时候，我们不会想到西方国家的保安服务合同商，如（美国的具有军事性的）黑水保安公司。实际上，中国的民营保安合同商所雇用的保安人员做的是看管停车场或类似的事情。另外还有人担任保镖。</p><p>因此，中国的民营保安合同商的业务更多的是跟私营部门的保安相关的，跟美国的黑水保安公司或俄罗斯的瓦格纳集团不是一回事。</p><p><strong>中国与西方保安合同商差距在哪里</strong></p><p><strong>问：</strong>就在海外从事保护中国利益的中国民营保安合同商的专业性而言，在你看来，中国的民营保安合同商和西方的合同商最明显的差距在哪里？究竟有什么重大障碍使中国的合同商的专业性难以发展？</p><p><strong>答：</strong>在我看来，中国的民营保安公司跟西方的公主要在四个方面有明显差距。第一位的而且是主要的差距是在公国公司的组织和总体能力方面。比如，假如跟西方的民营保安公司或民营保安军事公司相比，中国的民营保安公司有一个内在的问题，这就是，在中国，政府对武器有垄断。这就大大降低了中国民营保安公司专业性水平，大大缩减了中国民营保安公司能够提供的服务范围。</p><p>中国跟西方民营保安公司的第二个重大差别是我所说的核心能力，尤其是训练水平，中国的公司落后于俄罗斯的公司，也落后于西方的公司。从很多意义上说，这也是因为中国的保安公司从来没有参与过最初的军事冲突，也没有权利像西方的公司一样可以拥有武器。</p><p>阻碍中国的民营保安公司提升其专业性的第三个主要障碍是中国政府的立场。我认为，就我们所讨论的民营保安行业而言，现在有两个模式。一个是带有军事性的西方模式，主要是美国、英国、南非采用。再一个模式是俄罗斯模式。</p><p>在我看来，中国政府仍在斟酌究竟要采用什么模式更适合中国。中国方面的这种举棋不定或谨慎小心的做法阻碍了中国的民营保安合同商进一步发展他们的技能。</p><p>我觉得非常有趣的第四点是我所说的人力资源方面。在美国有很多在中东等地方参加过军事冲突的退伍军人，还有很多人作为国民警卫队成员参加过地方的军事行动。因此，这些人娴熟于使用武器。但在中国没有这样的人力资源。</p><p>因此，我认为这四方面的障碍使中国的民营保安公司达不到西方公司的专业标准。</p><p><strong>中国民营保安业现状与未来展望</strong></p><p><strong>问：</strong>为什么符合国际标准规范的或加入民营保安供应商国际行为守则的中国民营保安合同商如此之少？</p><p>答：这方面的主要问题牵涉海外运营领域的立法挑战和立法缺失。有关专家经常指出，有关中国的民营保安合同商的司法框架主要是涵盖在中国领土上的运营和活动。与此同时，中国在政府层级上不参与民营保安供应商国际行为守则（the International Code of Conduct for Private Security Providers ）也给中国民营保安合同商扩大海外业务造成了障碍。</p><p>国际间有100个民营军事和保安公司以及七个国家（澳大利亚，加拿大，挪威，瑞典，瑞士，英国和美国）认同该行为准则。据报道，2017年，由18个地方保安协会组成的中国保安协会在评估中国的民营保安合同商，以便准备合格公司白名单，那些公司被认为可以从事海外业务。但自那时以来没有进一步的发展的消息。</p><p>另外，中国民营保安合同商在佩戴武器和技术准备方面受限制，这就基本上使它们没有资格符合民营保安供应商国际行为守则的规范标准。</p><p>关于中国民营保安合同商未来发展的问题，最后我想补充两点。一个是中国民营保安公司领域出现新的发展趋势和新的业务范围。我在这所指的主要是网络空间的保安。中国的国家元首在很多场合强调了这个领域。我们需要观察民营保安公司如何参与这个领域的业务，如何在这个领域工作。</p><p>第二个是中国民营保安合同商在中亚和撒哈拉以南地区的业务发展。为什么要特别指出中亚和撒哈拉以南地区？因为中国在那些地区的投资明显超过其他国家。因此在这两个地区，中国的人员和金融财产安全和稳定面临可见的威胁。因此，这可能使中国增加在那些地区的民营保安公司的使用。</p>
