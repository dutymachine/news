[谴责北京意图改变现状，台湾坚称台湾海峡是“国际水域”](https://www.voachinese.com/a/taiwan-strait-is-an-international-waterway-taipei-says-in-rebuff-to-china-20220614/6616934.html)
------

<div><i>Tue, 14 Jun 2022 14:41:22 GMT</i></div><img src="https://images.weserv.nl?url=gdb.voanews.com/09470000-0a00-0242-3fc2-08da4df46fe0_cx0_cy17_cw86_r1_s_w900.jpg" width="100%"><div><small style="color: #999;">美国海军伯克级导弹驱逐舰“桑普森”号穿越台湾海峡。（2022年1月25日）</small></div><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">台湾的外交部星期二（</span><span style="color:#000000">6</span><span style="color:#000000">月</span><span style="color:#000000">14</span><span style="color:#000000">日）表示，台湾海峡是国际水域，适用国际法“公海自由”原则；台湾也理解并支持美国在台海自由航行的任务对促进地区和平稳定带来的助益。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">台湾外交部的声明被视为针锋相对地驳斥和反击北京外交部前一天有关国际海洋法上并无“国际水域”一说、中国对台海享有主权和管辖权说法。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“台湾海峡是国际水域，属于中华民国台湾领海范围以外的水域均适用国际法的‘公海自由’原则，”台湾外交部发言人欧江安在星期二的例行记者会上说。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">台湾海峡是分割中国大陆与台湾的一条狭长水道。自从国民党部队在中国的内战中败给中国共产党的武装并迫使中华民国政府</span><span style="color:#000000">1949</span><span style="color:#000000">年迁移台湾之后，台湾海峡经常成为两岸武装对峙、甚至爆发零星冲突的地点。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">自从中共</span><span style="color:#000000">1949</span><span style="color:#000000">年在北京成立中华人民共和国以来，台海两岸一直隔海分离分治、互不隶属。但是北京始终声称拥有对台湾的主权，而且坚持表示不排除诉诸武力实现两岸的统一。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">随着中国经济和军力的发展，北京对台湾的政治外交打压和军事恫吓也有增无减。这一局面引起与台湾理念相同的美国及其欧洲盟国的警觉与不满。美日欧对台湾的支持力度也越来越大。美国及其盟友还经常不顾北京的抗议与反对，派遣军舰穿越台湾海峡，以行动力挺台湾，同时对北京展示威慑与警告。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“</span><span style="color:#000000">台湾向来尊重外国船舶在台湾海峡任何符合国际法的行动，包括无害通过；台湾理解并支持美国自由航行任务对促进区域和平与稳定所带来的助益，</span><span style="color:#000000">”</span><span style="color:#000000">台湾外交部发言人欧江安说。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">彭博社日前引述匿名人士的话报道说，中国军方官员最近几个月在与美方官员的会晤中多次声称，台湾海峡不是国际水域，言外之意是美国军舰未经中国允许不能驶入台湾海峡。中方官员的这一说法，引起拜登政府内部的关切。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">中国外交部发言人汪文斌在星期一举行的例行记者会上更加明确地表达了北京对台湾海峡的立场和看法。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“台湾海峡最窄处约</span><span style="color:#000000">70</span><span style="color:#000000">海里，最宽处约</span><span style="color:#000000">220</span><span style="color:#000000">海里。根据《联合国海洋法公约》和中国国内法，台湾海峡水域由两岸向海峡中心线延伸，依次为中国的内水、领海、毗邻区和专属经济区，”汪文斌说。“中国对台湾海峡享有主权、主权权利和管辖权，同时也尊重其他国家在相关海域的合法权利。” </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">汪文斌还整个儿否认了“国际水域”的法理概念。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“国际海洋法上根本没有‘国际水域’一说。有关国家声称台湾海峡是‘国际水域’，意在为其操弄涉台问题、威胁中国主权安全制造借口。中方对此坚决反对，”汪文斌说。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">北京一直以来都坚持认为，台湾海峡是其专属经济区一部分，因此外国军舰在这一海域的活动应该受到限制。虽然中国常常对美国军舰穿越台海表达不满与抗议，有关水域的法理地位问题从未成为美中官员对话中的一个正式话题。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">台湾外交部发言人欧江安将北京对台湾海峡的说法斥为“谬论”。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“中国政府漠视台湾主张，刻意扭曲国际法规则，将台湾海峡矮化为其专属经济海域。中方企图并吞台湾的野心昭然若揭，外交部无法接受并予以谴责，”欧江安说。“中方此举无助于降低区域紧张情势。台湾将持续与理念相近国家紧密合作，共同捍卫以规则为基础的国际秩序，促进印太区域的和平、稳定与繁荣。” </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">台湾执政的民进党星期二也发表声明，痛批北京当局“执意将台湾海峡‘内海化’”。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“这种意图颠覆国际海洋秩序的恶意主张，只会激发有关国家的挞伐和激烈对抗，升高区域紧张态势，”民进党的声明说。“民进党要求，中共当局必须悬崖勒马，切勿一错再错，造成不可挽回的后果。” </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">台湾民进党籍立委王定宇也指责北京对台湾海峡表达的主张“鲁莽、不文明”，并且“已踩到红线，让人无法接受及容忍”。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“中国这样的主张并非突发奇想、临时起意，而是有计划性的，要排除国际自由民主联盟船舰通过这些法定的公海，组织国际船舰经过，让中国发动武力扩张时，能将其定义为内部行为，阻止国际用侵略的角度来看待，”王定宇说。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">彭博社在报道中国军方官员在与美方会晤时一再谈及台海不是国际水域的立场时，也曾寻求美国国防部的看法与反应。 </span></p><p style="background-color:#ffffff; line-height:1.38"><span style="color:#000000">“美国将继续在国际法允许的地方飞越、航行和运作，而这也包括穿越台湾海峡，”美国国防部发言人马丁</span><span style="color:#000000">∙</span><span style="color:#000000">梅纳斯（</span><span style="color:#000000">Martin Meiners</span><span style="color:#000000">）以电邮的方式回复彭博社的查询说。</span></p>
