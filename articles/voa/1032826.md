[耐克跑步应用程序将跑离中国](https://www.voachinese.com/a/nike-says-to-end-run-club-app-in-china/6608805.html)
------

<div><i>Wed, 08 Jun 2022 21:00:52 GMT</i></div><img src="https://images.weserv.nl?url=gdb.voanews.com/38A515E4-1C69-42DF-94C4-CA76D9F2BAB5_r1_s_w900.jpg" width="100%"><div><small style="color: #999;">资料图：一名妇女走过北京的一家耐克产品专卖店。 （2021年3月25日）</small></div><p>全球最大的运动服装品牌耐克(Nike)星期三(6月8日)表示，将从7月开始停止其旗下广受欢迎的应用程序Nike Run Club (耐克跑步俱乐部，NRC)在中国大陆地区的运营。这是美国品牌对中国市场做出的最新服务调整。</p><p>耐克星期三早上通过Nike Run Club应用程序向中国用户告知了这个决定，并且感谢他们一直以来的支持。</p><p>这份声明没有就停止运营给出解释。耐克一名发言人对路透社说，公司已计划未来向中国用户提供“增强型和本地化的解决方案。”</p><p>耐克推出的Nike Run Club应用程序能够让用户记录他们的跑步情况，并且邀请好友一起参与挑战。耐克在多个国家提供这个应用程序，在该公司盈利最佳的中国市场有超过800万用户。</p><p>耐克在声明中说：“感谢曾与Nike一起奔跑。”</p><p>“很遗憾地告知您，NRC APP将于7月8日停止您所在地区的运营。”耐克在告知中国用户的这个声明中说。</p><p>耐克对路透社表示，公司还将继续投入中国的运动市场。</p><p>公司发言人说：“我们正在中国打造一个针对中国市场的生态系统，尤其是针对这个地区独特的消费者需求，以更好地服务运动人士。”</p><p>这位发言人说： “我们未来将以增强型和本地化的解决方案服务中国的跑步用户。”</p><p>领英(Linkedin)和雅虎(Yahoo)等其他一些美国科技品牌过去一年也停止或减少了在中国大陆运营。他们给出的理由包括内容审查和越来越繁杂的合规要求。</p><p>中国在内容等领域对互联网公司内容加大管控，并且实施了个人信息保护法等新的法律。</p><p>美国电商巨头亚马逊(Amazon)上个星期也宣布将关闭旗下电子阅读器Kindle在中国的业务。不过亚马逊说，做出这个决定不是因为中国政府的压力或审查。</p><p>在中国社交媒体平台微博上，一些用户对耐克将停止Nike Run Club在中国大陆的运营表示遗憾和伤心。</p><p>一位用户表示已使用这个应用程序七年。还有一位用户则说，应用程序记录了他跑过的每一步，还附上三个泪流满面的表情符号。</p><p>耐克在给中国用户的告知声明中说，即日起将为有需要的跑步用户提供数据导出服务。</p><p>据中国媒体报道，中国大陆用户还能继续使用耐克在微信上的训练小程序Nike Training Club(耐克训练俱乐部，NTC)。这个在去年上线的小程序提供健身和瑜伽等训练课程，并且可以记录运动数据。</p><p>中国媒体报道说，耐克将在7月加速在中国市场的数字平台转型，包括升级Nike App、中文版的SNKRS App和耐克官网，以及加大对微信生态的投入和打造。</p><p><em>（本文依据了路透社的报道。）</em></p>
